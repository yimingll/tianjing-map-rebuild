# 周家酒楼任务线

## 📋 家族基本信息

**家族名称**：周家 - 百年酒楼世家
**出生概率**：5%（较为稀有）
**家族位置**：清河城内城东街,紧邻城中心广场
**家族经营**：天香楼，清河城最大的三层酒楼

**父亲**：周天行（凡人，42岁）
- 背景：周家长子，年轻时游历江湖，见多识广，后继承家业
- 性格：豪爽健谈，善于察言观色，人脉极广
- 技能：酒楼经营大师、江湖见闻、基础拳脚功夫
- 特点：认识各路江湖人士，包括散修、镖师、商人、说书人

**母亲**：柳婉儿（凡人，38岁）
- 出身：柳家钱庄之女，精通账务管理
- 性格：温婉贤淑，心思细腻，善于理财
- 技能：账务管理、烹饪、识人辨物
- 特点：对待客人体贴周到，酒楼口碑极佳

**家境**：富裕（年入500两银子）
**住所**：五进大院，前三进是"天香楼"酒楼，后两进是住宅

**酒楼特色**：
- 一楼：大堂，散座，平民百姓聚集地
- 二楼：雅间，江湖人士、商人常客
- 三楼：天字号包间，偶有修士光顾
- 特色：每日午后有说书先生"白眉老人"说书

---

## 🎯 任务线主题

**核心路线**：从听说书到结识江湖人物，见识修仙世界的边缘
**成长主题**：江湖见闻、人情世故、见识广博、机缘巧合
**特色玩法**：社交互动、说书剧情、神秘客人、信息收集
**情感线索**：父亲的江湖往事、白眉老人的身份、修士食客的秘密

  if (playerData.items.includes("父亲的玉佩")) {
    基础概率 += 0.05;
  }

  if (playerData.items.includes("培元丹") && playerData.flags.已服用培元丹) {

  }

  if (playerData.items.includes("淬体丹") && playerData.flags.已服用淬体丹) {
    基础概率 += 0.10;
  }

  // BUFF加成
  playerData.buffs.forEach(buff => {
    基础概率 += buff.灵根加成 || 0;
  });

  return 基础概率;
}
