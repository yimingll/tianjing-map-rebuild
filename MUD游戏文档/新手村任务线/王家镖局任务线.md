# 王家镖局任务线

## 📋 家族基本信息

**家族名称**:王家 - 四代镖师世家
**出生概率**:2%(稀有)
**家族位置**:清河城东门大街,紧邻官道
**家族经营**:王家镖局,走镖运货,保镖护院

**父亲**:王云龙(练气五层,42岁)
- 背景:年轻时是散修,因缘巧合加入镖局,娶了镖局千金
- 性格:豪爽仗义,武艺高强,江湖经验丰富
- 技能:王家枪法(精通)、轻身术、基础阵法、练气功法
- 外号:"云龙枪"王云龙,清河城第一高手

**母亲**:赵雨晴(练气二层,38岁)
- 出身:王家镖局独女,自幼习武
- 性格:外柔内刚,持家有道,武艺不凡
- 技能:王家剑法、账务管理、烹饪

**家境**:小康偏富(年入300两银子,但开销大)
**住所**:前院是镖局,后院是三进大宅,有演武场

**家族历史**:
- 太爷爷王振威:创立镖局,百年前的传奇镖师
- 爷爷王天雄:扩大镖局,结交三教九流
- 父亲王云龙:现任总镖头,是家族第一个修士

**镖局规模**:
- 镖师12人(其中3人是练气修士)
- 护院8人
- 伙计若干
- 走镖路线遍及三州六郡

---

## 🎯 任务线主题

**核心路线**:从习武强身到理解侠义精神
**成长主题**:武道传承、侠义之心、江湖风云
**特色玩法**:武术启蒙、走镖见闻、危机应对、侠义抉择
**情感线索**:父亲的期望、母亲的关怀、江湖的残酷与温暖

---

#### 1岁 - 镖局的小少爷

**任务类型**:剧情观看
**触发条件**:角色创建完成,分配到王家

**剧情内容**:
```
【旁白】
清河城东门大街,王家镖局。

这是清河城最大的镖局,门口两只石狮威风凛凛。
"王家镖局"四个大字,是当朝状元题写。

今天,镖局格外热闹。

因为,总镖头王云龙的儿子出生了!

【前院】
镖师们放下刀枪,齐聚院中。

"恭喜总镖头!"
"镖局后继有人了!"

王云龙一身劲装,满脸喜色,给每个镖师发赏钱。

【后院】
产房中,母亲赵雨晴虚弱地抱着襁褓中的你。

【王云龙】
父亲大步走进,看到你,眼中满是柔情:
"夫人辛苦了。"

他小心翼翼地抱起你,粗糙的手掌托着你的小脑袋。

【赵雨晴】
"相公,给孩子起个名字吧。"

【王云龙】
父亲沉吟片刻:
"就叫...青锋吧。"

"青出于蓝,锋芒毕露。"

"愿他能继承王家武道,行侠仗义,闯出一番名堂!"

【旁白】
镖局的锣鼓声响起。

这一天,整个清河城都知道:
王家镖局,添了个小少爷。

这就是你的家。
刀光剑影,侠骨柔情。
```

**无选择项**
**效果**:
- 记录出生地:王家镖局
- 记录父母信息
- 开启王家专属任务线
- 镖局声望+10(出生即有影响力)

```
"好!是我王家的种!"

看到父亲的枪法,你...

选项A:兴奋地手舞足蹈,想模仿(天赋倾向)
```

  - 性格标签+1"沉稳"
  - 父亲点头:"是块练武的好料子。"

- **选项C**:
  - 母亲满意度+5
  - 母亲:"青锋还小,等大些再说。"
  - 性格标签+1"谨慎"
  - 但父亲略显失望

**效果**:
- 与武艺结缘
- 初步性格倾向形成

#### 3岁 - 第一次扎马步

**任务类型**:技能启蒙+选择
**触发条件**:自动触发

**剧情内容**:
```
【旁白】
你三岁了,父亲决定开始教你习武。

今天,是你人生第一堂武术课。

【演武场】
清晨,父亲把你带到演武场。

【王云龙】
"青锋,从今天开始,爹教你练武。"

"但是——"

他严肃地看着你:

"练武很苦。"

"风吹日晒,冬练三九,夏练三伏。"

"你准备好了吗?"

你用力点头。

【王云龙】
"好!先从最基础的开始——扎马步!"

父亲示范:双腿分开,屈膝下蹲,如骑马状。

"来,照着爹的样子做。"

你模仿父亲的姿势。

一开始还行,但很快腿就开始发抖。

"撑住!男子汉,要能吃苦!"

汗水流进眼睛,腿越来越酸...

---

【互动选择】
腿实在太酸了,你...

选项A:咬牙坚持,绝不放弃!(坚韧)

选项B:告诉父亲太累了,能否休息?(诚实)

选项C:偷偷站起来一点点(取巧)
```

**选择结果**:
- **选项A**:
  - 你坚持到最后一刻才倒下
  - 父亲欣慰地抱起你:
    "好样的!不愧是我的儿子!"

  - 获得特质"坚韧不拔"

- **选项B**:
  - 父亲点头:"累了就说,不要硬撑伤了根基。"
  - 获得特质"诚实"
  - 父亲满意度+10
  - 父亲教你正确的发力方式,以后训练事半功倍

- **选项C**:
  - 父亲立刻发现:
    "青锋!练武最忌偷奸耍滑!"
  - 父亲让你重新扎一次
  - 父亲满意度-5
  - 但获得特质"机灵"
  - 学会观察和应变

- 学会基础马步

#### 4岁 - 认识江湖

**任务类型**:学习+社交
**触发条件**:自动触发

**剧情内容**:
```
【旁白】
你四岁了,已经可以在镖局里走动。

镖局经常有各色人等来往,你第一次见识到江湖的多彩。

【前院】
今天,镖局接了一趟镖。

客人是个书生模样的年轻人。

【年轻书生】
"总镖头,在下要去青州,路途不太平,想请镖局护送。"

【王云龙】
"放心,王家镖局,信誉第一!"

你躲在柱子后面偷看。

那书生看到你,温和一笑:
"这是令郎?"

【王云龙】
"是,犬子青锋。青锋,出来见礼!"

你乖巧地走出来,行礼:"见过先生。"

【年轻书生】
书生拿出一本小册子:
"小公子年纪虽小,但礼数周全。这本《三字经》送你。"

你收下书,感谢。

---

【后续事件】
接下来几天,你见到了各种各样的人:

- 富商李老爷:雇镖局运送货物
- 瘦削老者:寡言少语,眼神锐利(疑似修士)
- 江湖侠客:豪爽大方,与父亲称兄道弟
- 江洋大盗(被抓):凶神恶煞,破口大骂

父亲晚上和你聊天:

【王云龙】
"青锋,今天看到那么多人,有什么想法?"

---

【互动选择】
你说...

选项A:"爹,我想学武功,保护好人,惩罚坏人!"(侠义)

选项B:"爹,江湖好复杂,我有点害怕。"(谨慎)

选项C:"爹,那个老者是修士吗?我能修炼吗?"(好奇)
```

**选择结果**:
- **选项A**:
  - 父亲大喜:
    "好!这才是我王家的血脉!"
  - 父亲讲述王家祖训:"行侠仗义,扶危济困"
  - 获得特质"侠义之心"
  - 父亲满意度+20
  - 镖师们对你刮目相看

- **选项B**:
  - 父亲摸你的头:
    "害怕是正常的。"
    "江湖险恶,但只要心中有正气,就不用怕。"
  - 获得特质"谨慎小心"
  - 母亲满意度+15
  - 父亲传授一些江湖经验

  - 父亲一愣:
    "你怎么知道他是修士?"
  - 你:"我感觉他不一样。"
  - 父亲露出欣赏的眼神:
    "你的感知很敏锐。"
    "没错,他是筑基期修士,来我们镖局托镖。"
  - 获得特质"感知敏锐"
  - 悟性+2
  - 父亲开始偶尔讲修仙界的事

**效果**:
- 认识江湖百态
- 性格进一步成形

**任务类型**:技能学习+家族传承

**剧情内容**:
```

你五岁了,父亲开始正式传授你王家枪法。

【王云龙】
"青锋,这是爹特意为你打造的练习枪。"

"从今天起,爹教你王家枪法。"

【王云龙】
"王家枪法,传承四代,共有十二式。"

"来,跟着爹练!"

你模仿父亲的动作。

一遍,两遍,十遍,百遍...

汗水浸透衣衫。

---

【三个月后】

你已经能勉强打出第一式的样子。

【王云龙】
"青锋,悟性不错!"

"但是,枪法不是死练就能成。"

"要用心体会其中的韵味。"

"青龙出水,讲究的是'势如破竹,一往无前'。"

"你现在的枪,有形无神。"

---

【互动选择】
为了练好枪法,你...

选项A:加倍苦练,每天多练一个时辰(勤奋)

选项B:仔细琢磨父亲的话,试图理解"神韵"(悟性)

选项C:请教镖局的枪法高手镖师老张(虚心)
```

**选择结果**:
- **选项A - 加倍苦练**:
  - 你每天练到天黑
  - 母亲心疼你,父亲却很欣慰
  - 一个月后,枪法熟练度大增
  - 体质+2
  - 获得特质"勤学苦练"
  - 王家枪法 Lv.1 → Lv.2

- **选项B - 理解神韵**:
  - 你反复观察父亲的演示
  - 某天清晨,你忽然有所领悟
  - 一枪刺出,竟带出破空声!
  - 父亲震惊:"青锋,你...你领悟了枪意雏形!"
  - 悟性+3
  - 获得稀有特质"枪道天赋"
  - 王家枪法 Lv.1 → Lv.3
  - 父亲满意度+25

- **选项C - 虚心求教**:
  - 老张是镖局枪法第二高手
  - 他耐心指点你的细节
  - "小少爷,枪头要稳,腰马要合..."
  - 在老张的指导下,你进步很快
  - 获得特质"虚心好学"
  - 王家枪法 Lv.1 → Lv.2
  - NPC"镖师老张"好感+30

**效果**:
- 学会王家枪法第一式
- 武艺启蒙
- 体质/悟性成长

#### 6岁 - 镖局遇险

**任务类型**:危机事件+勇气考验
**触发条件**:自动触发

**剧情内容**:
```
【旁白】
你六岁这年,镖局发生了一件大事。

"有人劫镖!快起来!"

【母亲】

"青锋,不要怕!"

【前院】
十几个蒙面黑衣人冲进镖局。

"交出镖银,饶你们不死!"

【王云龙】
父亲一身劲装,提枪而出:
"想劫王家的镖?问过我的枪没有!"

双方激战!

父亲一杆银枪,如龙出海,挡住七八个黑衣人。

镖师们也拿起兵器,与黑衣人搏杀。

但黑衣人中,有几个是练气修士!

镖局渐渐不敌。

【关键时刻】
一个黑衣人突破防线,冲向后院!

他看到了你!

"抓住小的,逼王云龙就范!"

黑衣人向你冲来!

母亲挡在你面前,拔剑相抗。

但她只有练气二层,不是对手!

眼看黑衣人就要得手——

---

【互动选择】
这个危急时刻,你...

选项A:抓起床边的小枪,冲上去帮母亲!(勇敢)

选项B:大声呼救:"爹!救命!"(求助)

选项C:保护妹妹,把她藏起来(责任)
  (注:如果你有妹妹的话,没有则此选项变为:躲到床下,瑟瑟发抖)
```

**选择结果**:

**选项A - 勇敢战斗**:
```

"不许伤我娘!"

你使出刚学会的第一式:青龙出水!

枪尖刺向黑衣人。

【黑衣人】
"小鬼找死!"

他一掌拍向你。

但就在这时——

嗖!

一道银光闪过!

【王云龙】
父亲赶到!一枪逼退黑衣人!

"敢动我儿子!"

父亲暴怒,枪法大开大合,黑衣人节节败退。

【战斗结束后】

黑衣人撤退。

父亲抱起你,又气又心疼:

"青锋!你个臭小子!才多大就冲上去!"

"要是有个三长两短,让爹怎么办!"

但他眼中满是骄傲。

母亲也哭了:
"青锋,你吓死娘了..."

【后续影响】
- 镖局所有人都知道了小少爷的勇气
- 镖师们对你刮目相看
- 父亲满意度+30
```

```
父亲听到,立刻冲回后院!

一枪挑飞黑衣人的兵器,救下你和母亲。

父亲摸你的头:
"青锋,遇到危险,呼救是对的。"
```

**选项C - 保护妹妹**:
```
(如果有妹妹)
```

- 王家镖局是稀有家族(2%概率)

- 与李家铁匠铺的区别:
  - 李家偏重"炼器技艺",王家偏重"武道侠义"
  - 李家是"父亲的遗憾",王家是"父亲的传承"
  - 李家更内敛,王家更外放
- 情感深度和选择丰富度与李家保持一致
- 特殊NPC(李明、白衣修士)为后续剧情埋下伏笔
