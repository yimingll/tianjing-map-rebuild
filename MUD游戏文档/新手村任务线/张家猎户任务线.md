# 张家猎户任务线

## 📋 家族基本信息

**家族名称**：张家 - 祖传猎户世家
**出生概率**：20%（较常见）
**家族位置**：清河城外城南区，靠近北城门
**家族经营**：世代狩猎为生，售卖兽皮、兽肉、药材

**父亲**：张虎（凡人武者，45岁）
- 背景：三代猎户传人，十五岁便独自入山狩猎
- 性格：豪爽直率，义气深重，疼爱子女
- 技能：射箭精准、追踪高手、精通陷阱、凡人武艺（外家拳法）
- 特点：曾在深山救过一位受伤修士，获赠《呼吸吐纳法》

**母亲**：林巧儿（凡人，39岁）
- 出身：附近村庄猎户之女
- 性格：温柔坚韧，持家有道，心灵手巧
- 技能：剥皮鞣制、缝补衣物、识别草药、烹饪野味

**家境**：中等（年入80两银子，但不稳定）
**住所**：两进院落，前院晾晒兽皮，后院住人，有小仓库存放猎具

---

## 🎯 任务线主题

**核心路线**：从听深山传说到独立面对自然与危险
**成长主题**：野性成长、生存智慧、敬畏自然、勇气担当
**特色玩法**：追踪学习、射箭训练、深山历险、妖兽奇遇
**情感线索**：父亲的传承、深山的神秘、生死的洗礼、成长的代价

---

#### 1岁 - 猎户之家的新生

**任务类型**：剧情观看
**触发条件**：角色创建完成，分配到张家

**剧情内容**：
```
【旁白】
清河城外城南区，靠近北城门。

这里住着一户猎人家。

深夜，一声婴儿的啼哭打破寂静。

你出生了。

父亲张虎刚从深山回来，一身风尘仆仆。
他放下肩上的猎物，大步走进产房。

粗糙的大手小心翼翼地抱起你，豪爽的汉子眼眶竟有些泛红。

【张虎】
"巧儿，是个小子！哈哈，咱张家又添了个男丁！"

【林巧儿】（虚弱地笑）
"夫君，给孩子起个名字吧。"

【张虎】
父亲看着襁褓中的你，沉吟片刻：

"就叫猛子吧。张猛。"

"愿他像小老虎一样，虎虎生威，在这世上立足！"

院子里，挂着一张还在滴血的虎皮——那是父亲为庆祝你出生，
特意进深山猎回来的。

【旁白】
你还不懂这意味着什么。

你只知道，这个家有父亲强壮的臂膀，有母亲温柔的怀抱。

还有院子里那张虎皮，在月光下泛着血色的光。

这就是你的家。
猎户之家。
```

**无选择项**
**效果**：
- 记录出生地：张家猎户
- 记录父母信息
- 开启张家专属任务线

#### 7岁 - 第一次独自狩猎

**任务类型**：重要里程碑+技能检定
**触发条件**：自动触发

**剧情内容**：
```
【旁白】
你七岁了，射箭已经很准了。

今天，父亲说了一句让你心跳加速的话。

【张虎】
"猛子，今天你要自己打猎。"

这是你人生中第一次真正的狩猎。

【狩猎过程】

你悄悄进入林中。

回忆父亲教的所有技巧：
- 脚步要轻
- 顺风而行
- 观察鸟的动静
- 听草丛的声音

忽然，你看到了——

一只野鸡，正在啄食地上的虫子。

距离你二十步。

你蹲下身，慢慢拉弓...

---

【关键检定】
射箭的关键是...

选项A：屏住呼吸，稳定手臂（稳定性）

选项B：瞄准要害，一击毙命（准确性）

选项C：感受风向，调整角度（悟性）
```

**选择结果**：

**选项A - 稳定性**：
```
你屏住呼吸。

手臂纹丝不动。

心跳慢了下来。

世界仿佛静止。

只有那只野鸡，在你眼中。

放箭！

咻——

正中！

野鸡扑腾几下，不动了。

你成功了！

【后续】
父亲从树后走出，眼中满是欣慰：

"好！射得稳！"

"这是你人生第一猎。"

【获得】
- 技能"射箭 Lv.1" 提升至 Lv.3
- 获得特质"沉稳之心"
- 体质+1
- 父亲满意度+20
- 自己猎回的野鸡（晚上加餐）
```

**选项B - 准确性**：
```
你瞄准野鸡的头部。

那是最致命的部位。

但也是最小的目标。

赌一把！

放箭！

咻——

正中头部！

野鸡连挣扎都没有，当场毙命。

【后续】
父亲惊讶地走过来：

"好小子！居然敢瞄准这么小的目标！"

"而且还射中了！"
```

## 🎮 游戏性分析

### 重玩价值：极高
- 45+个选择点
- 救不救小青是完全不同的体验
- 不同性格影响NPC反应
- 隐藏事件触发条件多样

### 情感深度：优秀
- 父子情深贯穿始终
- 小青线增加温情
- 深山遇险生死与共
- 离别催人泪下

### 平衡性：良好
- 有灵根/无灵根都有后续
- 不同选择无绝对优劣
- 箭道天赋可补救无灵根
- 小青是情感线，非必须
