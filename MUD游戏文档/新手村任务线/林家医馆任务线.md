# 林家医馆任务线

## 📋 家族基本信息

**家族名称**：林家 - 三代医者世家
**出生概率**：3%（稀有）
**家族位置**：清河城外城东区,靠近城隍庙
**家族经营**：林家医馆,善治各种疑难杂症

**父亲**：林济世（练气四层,45岁）
- 背景：青云医仙门外门弟子,因医术高超被派驻凡间行医
- 性格：慈悲为怀,医者仁心,悬壶济世
- 技能：医术大师、炼丹入门、基础修炼

**母亲**：苏婉儿（练气一层,38岁）
- 出身：清河城苏家,曾是林济世救治的病患,后结为连理
- 性格：温柔善良,心思细腻,擅长药膳
- 技能：药膳制作、护理、识药

**家境**：小康偏上（年入150两银子,但常免费施医）
**住所**：三进院落,前院是医馆,中院制药,后院住人

---

## 🎯 任务线主题

**核心路线**：从观察行医到理解医者仁心,从认药学药到初步掌握炼丹之道
**成长主题**：悬壶济世、医道传承、仁心仁术、生命敬畏
**特色玩法**：医道启蒙、药材认知、炼丹学习、诊疗观摩
**情感线索**：父亲的慈悲心、母亲的感恩情、患者的信任、医者的责任

---

**任务类型**：剧情观看
**触发条件**：角色创建完成,分配到林家

**剧情内容**：
```

窗外是城隍庙的钟声,屋内是父亲温和的声音。

你出生在这样一个充满生机与希望的地方。

母亲苏婉儿躺在床上,虚弱但欣慰地看着襁褓中的你。

父亲林济世小心翼翼地抱起你,在你额头轻轻一点。

一股温暖的力量流入你的身体——那是父亲为你种下的长生护命诀基础。

【林济世】
"婉儿,咱们的孩子很健康,体质也不错。"

【苏婉儿】
"夫君,给孩子起个名字吧。"

父亲沉思片刻,看着窗外的晨曦:

"就叫星辰吧。星辰点点,照亮黑暗。

【旁白】

你只知道,药香的温暖,父母的爱,和屋外病患感激的声音。

这就是林家医馆。

**无选择项**
**效果**：
- 记录出生地：林家医馆
- 记录父母信息

```

"这是人参,大补元气。"
"这是黄芪,补气升阳。"
"这是当归,补血活血。"
"这是川芎,活血行气。"

"星辰,药材分四气五味。"

  - 获得"药材鉴别 Lv.1"

  - 正确率75%
  - 父亲:"记忆力很好,是学医的好苗子。"

母亲在一旁看着,心疼地想帮忙,却被父亲拦住。

父亲仔细检查:

【林济世】
"嗯,重量都对。不错,星辰第一次抓药就这么准。"

  - 父亲满意度-8
