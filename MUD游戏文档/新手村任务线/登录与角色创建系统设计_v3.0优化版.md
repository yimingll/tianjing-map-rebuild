> **设计版本**: v3.0 世界观融合版
> **设计日期**: 2025-01-14
> **核心理念**: "人生电影模式" + "深度世界观融合" + "真实烟火气" + "多元分支"
>
> **v3.0特色**: 完全融合最新世界观设定，包括：
> - 人物志中的真实NPC
> - 5300年历史脉络
> - 详细的宗门组织架构
> - 当前局势的矛盾冲突
> - 真实的市井烟火生活

---

## 一、设计理念

### 1.1 核心目标

**让玩家在角色创建阶段就沉浸在青玄界中**

不是简单地"选属性、定职业"，而是：
- 体验一段完整的15年人生
- 了解青玄界的世界观
- 理解修士与凡人的区别
- 感受真实的成长历程
- 做出影响一生的关键选择

### 1.2 设计原则

✅ **世界观一致性** - 所有内容必须符合最新世界观设定
✅ **烟火气十足** - 既有修仙宏大叙事，也有凡人柴米油盐
✅ **选择有意义** - 每个选择都影响最终结果
✅ **重玩价值高** - 不同选择看到不同内容
✅ **时间成本控制** - 完整流程15-20分钟
✅ **可跳过机制** - 老玩家可以快速通过

## 二、时代背景设定

### 2.1 当前时间

**青玄历 5300年，春三月**

这是一个：
- 表面和平，实则暗流涌动的时代
- 距上次正魔大战已300年
- 怪事频发的时代（灵脉枯竭、遗迹频现、天象异常）
- 大劫将至的前夕

### 2.2 玩家出生地

**中州 · 清河城 · 李家小院**

**清河城背景**（参考《卷一·天下形胜》）：
- 中州第一大城
- 人口：凡人200万，修士3万
- 特点：商业繁华，修士众多，坊市林立
- 标志建筑：清河大桥、真君祠、灵宝塔

**李家小院位置**：
- 外城南区（平民区）
- 靠近东市（粮食市场）
- 胡同深处，三进小院
- 邻里：王婆婆（卖菜）、张铁匠（打铁）、赵秀才（教书）

### 2.3 时代氛围

玩家成长的这15年（5300-5315年），正是：
- 正道内部矛盾加剧
- 魔道暗中招兵买马
- 散修处境越来越难
- 灵脉枯竭引发恐慌
- 遗迹频繁出现，争夺激烈

**玩家会在成长过程中感受到这些变化**

## 三、家族背景系统

### 3.1 清河城十大家族（随机出生）

**设计理念**：
- 所有玩家都出生在清河城，但家族不同
- 家族背景**不提供初始属性加成**，保证公平性
- 家族背景影响成长事件和剧情走向
- **15岁测灵后**，根据家族赠送不同的特殊物品

**家族分布**：
```
清河城外城南区（平民区）：
├─ 李家铁匠铺（30%）
├─ 孙家药圃（25%）
├─ 张家猎户（20%）
└─ 陈家孤儿院（0.5%）

清河城外城东区（商业区）：
├─ 钱家布庄（15%）
└─ 周家酒楼（5%）

清河城内城（富人区）：
├─ 赵家书院（8%）
├─ 白家老宅（1.5%）
└─ 林家医馆（3%）

清河城特殊区域：
└─ 王家镖局（2%）
```

### 3.2 家族赠礼说明

**赠礼时机**：15岁测灵结束后，回家告知结果时触发

**赠礼品质分级**：
- 一品：最基础的修炼物品
- 二品：有一定价值，对前期有帮助
- 三品：珍贵物品，可用到筑基期

**特殊说明**：
- 白家（没落修仙家族）赠礼最珍贵，但也伴随家族仇敌威胁
- 孤儿院赠礼虽不豪华，但情感价值最高
- 商贾家族赠送资金最多，但需要自己购买
- 各家族赠礼都有其特色和用途，无绝对优劣


### 3.2 父母人设系统

根据家族背景，父母有不同的教育方式，影响性格养成：

**父亲教育风格**：
- **严父型**：要求严格，犯错必罚 → 性格坚毅
- **慈父型**：宠溺孩子，有求必应 → 性格温和但可能软弱
- **放养型**：自由成长，不多管 → 独立但可能叛逆
- **期望型**：望子成龙，压力大 → 上进但可能压抑

**母亲性格类型**：
- **温柔型**：嘘寒问暖，无微不至
- **坚强型**：里外一把手，性格刚强
- **病弱型**：身体不好，需要照顾（触发特殊剧情）
- **智慧型**：知书达理，常教导人生道理

## 五、角色生成系统

### 5.1 属性计算公式

**基础属性**：根据15年选择累计
```
力量: 家族背景 + 年龄事件选择 + 特殊奖励
体质: 同上
敏捷: 同上
悟性: 同上
运气: 同上
魅力: 同上
```

**修炼倾向**：
```
炼器亲和: 铁匠家族+10, 跟父亲学艺+10, 等
炼丹亲和: 药农家族+10, 认药事件+5, 等
剑修亲和: 特定事件触发
儒修亲和: 书香门第+10, 跟赵夫子学习+10, 等
```

### 5.2 灵根生成算法

```
总积分 = Σ(15年所有选择的分数)

灵根品质概率:
- 天灵根: 总积分 > 1000 且运气 > 95 (0.1%)
- 伪灵根: 总积分 > 800 且运气 > 85 (0.5%)
- 单灵根: 总积分 > 600 且运气 > 70 (1%)
- 双灵根: 总积分 > 400 (10%)
- 三灵根: 总积分 > 200 (30%)
- 四灵根: 总积分 > 100 (50%)
- 五灵根/无: 总积分 <= 100 (8.4%)

灵根属性倾向:
- 根据11岁梦境和前期选择
- 金: 炼器/战斗倾向
- 木: 炼丹/治疗倾向
- 水: 智慧/灵活倾向
- 火: 攻击/激情倾向
- 土: 防御/稳重倾向
```

## 文档总结

这份v3.0优化版登录与角色创建系统设计完全融合了最新的世界观设定，包括：

1. **准确的时代背景**：青玄历5300年，山雨欲来的时代
2. **详细的地理位置**：清河城的街道、市集、邻里
3. **真实的NPC人物**：每个人都有完整的人生故事
4. **符合历史的背景**：参考大事年表的历史脉络
5. **浓郁的烟火气**：茶楼、说书人、街坊邻居
6. **深度的世界观融合**：正魔之争、遗迹频现、当前局势

**核心特色**：
- 不是"创建角色"，而是"经历一段15年的人生"
- 每个选择都有意义，影响最终的灵根和性格
- 有血有肉的家人朋友，让玩家真正产生情感连接
- 完美衔接14个世界观文档，保证一致性
- 说书人叙事风格，充满文化底蕴

**与旧版对比**：
- 世界观深度：从简单设定到14文档完全融合
- NPC立体度：从扁平工具人到有完整人生的角色
- 烟火气：从无到大量市井生活细节
- 叙事水平：从游戏化到文学化

这份设计为游戏提供了一个完整的、沉浸式的角色创建体验，
玩家将真正感受到这个世界的真实性和温度。
