# 新手村任务线总览

## 📖 系统概述

### 设计理念
- **年龄范围**：1岁-10岁（童年成长期）
- **触发时机**：玩家随机分配家族后自动开启
- **任务特点**：每个家族有独特的成长任务线
- **结局衔接**：10岁测灵大典后进入修仙世界

### 核心流程
```
创建角色 → 随机家族（10选1）→ 1-10岁成长任务线 → 10岁测灵大典 → 家族赠礼 → 进入修仙世界
```

---

## 🎯 年龄阶段划分

### 第一阶段：幼年期（1-3岁）
**主题**：家族认知、性格萌芽
- 主要内容：观察父母、家族环境熏陶
- 选择影响：性格倾向的初步形成
- 任务类型：剧情观看为主，少量简单选择

### 第二阶段：童年期（4-6岁）
**主题**：技能启蒙、天赋初显
- 主要内容：学习家族基础技能、展现天赋
- 选择影响：影响后续特殊事件触发
- 任务类型：互动增加，有简单的技能学习

### 第三阶段：少年期（7-9岁）
**主题**：成长抉择、关键事件
- 主要内容：面临重要选择、参与家族事务
- 选择影响：决定性格特质、人际关系
- 任务类型：分支剧情、重要抉择

### 第四阶段：测灵前夕（10岁）
**主题**：准备测灵、人生转折
- 主要内容：测灵大典准备、家族期望
- 选择影响：影响测灵剧情和家族赠礼场景
- 任务类型：高潮剧情、情感渲染

## 🏠 十大家族任务线概览

### 1. 李家铁匠铺（30%）- 炼器启蒙线
**核心路线**：从观察打铁到初步理解炼器之道
- **1-3岁**：铁匠铺的童年，火炉旁的记忆
- **4-6岁**：第一次握锤、认识材料、帮忙拉风箱
- **7-9岁**：打造第一件铁器、发现父亲的秘密
- **10岁**：测灵前，父亲传授炼器心得
- **特色事件**：发现父亲的修炼笔记、退宗真相

### 2. 孙家药圃（25%）- 灵药培育线
**核心路线**：从识药采药到理解灵药之妙
- **1-3岁**：药圃的四季、草药的气味
- **4-6岁**：认药材、学浇水、帮忙晾晒
- **7-9岁**：发现变异灵药、学习基础炼丹
- **10岁**：测灵前，母亲赠百年灵参
- **特色事件**：药圃发现灵药幼苗、意外治愈修士

### 3. 张家猎户（20%）- 深山历练线
**核心路线**：从跟随狩猎到独立面对危险
- **1-3岁**：听父亲讲深山故事、妖兽传说
- **4-6岁**：学习追踪、认识兽类、练习箭术
- **7-9岁**：第一次进山、救助妖兽幼崽
- **10岁**：测灵前，深山遇险与成长
- **特色事件**：妖兽幼崽成长、深山遗迹发现

### 4. 钱家布庄（15%）- 商道启蒙线
**核心路线**：从帮工算账到理解商道人情
- **1-3岁**：布庄的繁华、各色客人
- **4-6岁**：学珠算、识布料、接待客人
- **7-9岁**：参与商队、见识修仙坊市
- **10岁**：测灵前，父亲传授经商之道
- **特色事件**：遇到修士客人、坊市奇遇

### 5. 周家酒楼（5%）- 江湖见闻线
**核心路线**：从听说书到结识江湖人物
- **1-3岁**：酒楼的热闹、说书人的故事
- **4-6岁**：学待客、听江湖事、认识食客
- **7-9岁**：白眉说书人的指点、江湖秘闻
- **10岁**：测灵前，获得《江湖见闻录》
- **特色事件**：神秘食客、说书人的预言

### 6. 赵家书院（8%）- 儒学启蒙线
**核心路线**：从读书识字到理解浩然正气
- **1-3岁**：书香门第、诗书礼仪
- **4-6岁**：启蒙读书、练习书法、学习礼仪
- **7-9岁**：参加文会、展现才学、儒道感悟
- **10岁**：测灵前，祖父赠浩然正气符
- **特色事件**：书院奇遇、文道启迪

### 7. 白家老宅（1.5%）- 家族复兴线
**核心路线**：从了解家族荣耀到立志复兴
- **1-3岁**：听家族往事、感受没落氛围
- **4-6岁**：学习家传功法、发现祖传法宝
- **7-9岁**：遭遇仇家、誓要复兴家族
- **10岁**：测灵前，获得白家传承令
- **特色事件**：家族仇敌、祖辈秘密

### 8. 林家医馆（3%）- 医道启蒙线
**核心路线**：从观察行医到理解医者仁心
- **1-3岁**：医馆的药香、父亲的医术
- **4-6岁**：学认药、帮忙抓药、观摩诊治
- **7-9岁**：救治病人、学习炼丹、医道感悟
- **10岁**：测灵前，父亲赠九转还魂丹
- **特色事件**：治愈重症、修士求医

### 9. 王家镖局（2%）- 武道启蒙线
**核心路线**：从习武强身到理解侠义精神
- **1-3岁**：镖局的热血、父亲的威武
- **4-6岁**：扎马步、学枪法、练习基本功
- **7-9岁**：跟随走镖、遇到危险、侠义抉择
- **10岁**：测灵前，父亲传授王家枪法
- **特色事件**：走镖奇遇、江湖恩怨

### 10. 陈家孤儿院（0.5%）- 身世之谜线
**核心路线**：从孤儿成长到发现身世线索
- **1-3岁**：被陈老伯收养、孤儿院的温暖
- **4-6岁**：帮忙照顾弟妹、寻找身世线索
- **7-9岁**：神秘访客、护体玉佩的秘密
- **10岁**：测灵前，发现自己的特殊血脉
- **特色事件**：神秘来客、血脉觉醒

## 🎁 10岁测灵与家族赠礼

### 测灵大典流程
```
家族准备 → 前往测灵台 → 测试灵根 → 公布结果 → 返回家中 → 家族赠礼 → 告别亲人 → 进入修仙世界
```

### 家族赠礼系统
每个家族根据测灵结果（有灵根/无灵根）赠送不同物品：

|------|-----------|-----------|
| 李家铁匠铺 | 百炼精铁（三品）+炼器心得 | 打铁手艺+凡铁剑 |

| 张家猎户 | 青狼皮甲（二品）+猎弓 | 狩猎技艺+全套装备 |

## 📊 任务线设计原则

### 1. 公平性原则
- 所有家族任务线难度相当
- 不同家族给予不同类型奖励，但价值相近
- 无绝对优劣，只有风格差异

### 2. 情感深度原则
- 强化玩家与家人的情感联系
- 每个选择都有情感权重
- 离别时刻更有代入感

### 3. 重玩价值原则
- 10条任务线各具特色
- 稀有家族（白家1.5%、孤儿院0.5%）有独特剧情
- 不同选择导向不同结局

### 4. 衔接性原则
- 与后续修仙世界无缝衔接
- 家族赠礼物品在游戏中有实际作用
- 成长经历影响角色性格和对话选项

## 🔧 技术实现建议

### 任务数据结构
```javascript
{
  family: "李家铁匠铺",
  questline: [
    {
      age: 1,
      title: "铁匠铺的新生儿",
      type: "剧情观看",
      content: "...",
      choices: [],
      effects: []
    },
    {
      age: 5,
      title: "第一次握锤",
      type: "技能启蒙",
      content: "...",
      choices: [
        {text: "全力以赴", effect: "性格+坚毅"},
        {text: "量力而行", effect: "性格+谨慎"}
      ]
    }
    // ... 更多任务
  ]
}
```

### 触发逻辑
```javascript
// 分配家族后
player.family = randomFamily();

// 加载对应任务线
player.questline = loadFamilyQuestline(player.family);

// 按年龄自动触发
onPlayerAgeUp(age) {
  let quests = player.questline.filter(q => q.age === age);
  quests.forEach(q => triggerQuest(q));
}
```
