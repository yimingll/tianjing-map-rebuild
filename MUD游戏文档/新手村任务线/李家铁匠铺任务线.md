# 李家铁匠铺任务线

## 📋 家族基本信息

**家族名称**：李家 - 三代铁匠世家
**出生概率**：30%（最常见）
**家族位置**：清河城外城南区，靠近东市
**家族经营**：李家铁匠铺，专做农具、刀剑

**父亲**：李铁柱（练气三层，40岁）
- 背景：年轻时是玄天宗外门弟子，因意外道基受损被迫退宗
- 性格：沉默寡言，手艺精湛，对你寄予厚望
- 技能：炼器入门、打铁大师、基础修炼

**母亲**：王秀娘（凡人，35岁）
- 出身：清河城王家，普通商户之女
- 性格：温柔贤惠，持家有道，关心体贴
- 技能：持家、烹饪、账务管理

**家境**：小康（年入100两银子）
**住所**：三进小院，前院是铁匠铺，后院住人

---

## 🎯 任务线主题

**核心路线**：从观察打铁到初步理解炼器之道
**成长主题**：匠心传承、父子情深、修仙梦想
**特色玩法**：炼器启蒙、材料认知、工艺学习
**情感线索**：父亲的隐痛、母亲的关怀、家族的期望

---

## 🎮 游戏性分析

### 重玩价值：极高
- 多达40+个选择点
- 不同选择导向不同的角色性格
- 影响测灵概率
- 影响赠礼内容
- 影响后续剧情展开

### 情感深度：优秀
- 父子情深贯穿始终
- 每个选择都有情感权重
- 离别时刻催人泪下
- 玩家与角色产生强烈共情

### 平衡性：良好
- 有灵根/无灵根都有后续内容
- 不同选择无绝对优劣
- 稀有事件（孙师兄）有触发条件但不强求
- 保底机制（炼器天赋可补救无灵根）

## 💡 实现建议

### 技术要点
```javascript
// 核心数据结构
const playerData = {
  family: "李家铁匠铺",
  age: 1,
  traits: [], // 特质列表
  skills: {}, // 技能等级
  items: [], // 物品列表
  relationships: {
    father: 50,
    mother: 50,
    // ...
  },
  flags: {}, // 事件标记
  buffs: [], // BUFF列表
}

// 选择处理
function handleChoice(choiceId, option) {
  // 记录选择
  playerData.choices[choiceId] = option;

  // 应用效果
  applyChoiceEffects(choiceId, option);

  // 检查触发条件
  checkEventTriggers();
}

// 测灵系统
function testLinggen() {
  let probability = baseProbability;

  // 应用各种加成
  probability += getFatherBonus();
  probability += getBuffBonus();
  probability += getTraitBonus();

  // 随机判定
  return rollLinggen(probability);
}
```

### 剧情分支管理
建议使用树状结构管理分支：
```
主干（必经）
├─ 分支A（可选）
│  ├─ 子分支A1
│  └─ 子分支A2
├─ 分支B（隐藏）
│  └─ 特殊结局
└─ 汇合点
```

## ✨ 特色亮点

1. **父子情深**：整条任务线的情感核心
2. **炼器传承**：家族特色与修仙世界的完美结合
3. **多重结局**：有灵根、无灵根、特殊路线各有精彩
4. **成长感**：从1岁到10岁，能清晰感受角色成长
5. **选择自由**：40+个选择点，真正的"我的故事"
6. **隐藏深度**：多个隐藏事件和特殊路线待发掘
