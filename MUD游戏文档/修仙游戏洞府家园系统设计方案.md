# 修仙游戏洞府/家园系统设计方案
> **注**: 本方案为纯单机/买断制游戏设计，无充值系统

## 一、系统总览

### 核心设计理念
- **个人修仙基地**：玩家的"家"，所有生产活动的核心场所
- **整合所有职业系统**：炼丹房、炼器室、符道堂、阵法区等
- **灵脉品质系统**：影响修炼速度和生产效率
- **自由建造布局**：玩家自定义建筑位置和装饰
- **洞府等级成长**：从简陋洞府到洞天福地，再到独立小世界
- **社交展示功能**：好友访问，展示成就

>
> **重要**: 本文档定义洞府系统
>
> **依赖系统**: 货币系统、采集系统
>
> **参考标准**: 详见《【重要】文档统一标准.md》

- 8级：元婴期后可升级
- 9级：化神期后可升级
- 10级：渡劫期后可升级

## 三、洞府选址系统

### 1. 地理位置分类

#### 山洞类（最常见）
```
【青山洞府】- 位于普通山脉，灵气一般
【灵山洞府】- 位于灵山，灵气+20%
【仙山洞府】- 位于仙山，灵气+50%
【主峰洞府】- 位于山脉主峰，灵气+80%，风景绝佳
【山腹洞府】- 开凿山腹，隐蔽性强，灵气+30%
【悬崖洞府】- 悬崖峭壁，防御力强，灵气+40%
```

#### 海岛类（特殊）
```
【海岛洞府】- 位于海岛，水属性灵气+50%
【珊瑚岛】- 海底珊瑚，水灵气+80%，可养水灵兽
【仙岛】- 仙气缭绕，全属性灵气+100%
【龙宫】- 海底龙宫遗迹，水龙灵气，极其稀有
```

#### 云端类（高境界）
```
【云中仙府】- 漂浮云端，需要化神期，灵气+150%
【九霄云府】- 九天之上，需要炼虚期，灵气+200%
【星空洞府】- 星空深处，需要大乘期，灵气+300%
```

#### 地底类（特殊）
```
【地底洞府】- 地底深处，土属性+50%
【地火洞府】- 靠近地火，火属性+80%，炼器炼丹+20%
【地心洞府】- 地心深处，土火双属性+100%
【矿脉洞府】- 灵矿脉中，产出灵石+50%
```

#### 虚空类（仙人专属）
```
【虚空洞府】- 开辟虚空，独立空间，需要合体期
【次元洞府】- 次元夹层，完全隐匿，需要大乘期
【小千世界】- 独立小世界，可自主控制，需要渡劫期
```

### 3. 风水格局

#### 风水评级
| 风水等级 | 气运加成 | 机缘几率 | 特殊效果 |
|---------|---------|---------|---------|
| 绝地 | -20 | -50% | 容易走火入魔 |
| 凶地 | -10 | -20% | 修炼易出岔子 |
| 平地 | 0 | +0% | 无特殊效果 |
| 吉地 | +10 | +20% | 修炼顺畅 |
| 宝地 | +20 | +50% | 机缘增加 |
| 福地 | +50 | +100% | 天降福运 |
| 仙地 | +100 | +200% | 飞升几率+5% |

#### 风水要素
- **龙脉**：是否处于龙脉之上（大幅提升灵气）
- **明堂**：洞府前方是否开阔（影响视野和气运）
- **朝向**：洞府朝向（坐北朝南最佳）
- **水源**：是否临水（水为财，影响资源产出）
- **背靠**：是否背靠山峰（安全感，防御+20%）

## 四、洞府建筑系统（30+建筑）

### 核心建筑（必需）

#### 1. 洞府大殿（主建筑）
- **功能**：洞府控制中心，升级洞府、管理建筑
- **等级**：1-10级（对应洞府等级）
- **建造费用**：随洞府升级自动升级
- **特殊功能**：
  - 查看洞府信息（面积、灵气、建筑数）
  - 升级洞府等级
  - 空间扩展（消耗空间扩展符）
  - 洞府传送（回到洞府）

#### 2. 修炼密室
- **功能**：闭关修炼，修炼速度×1.5
- **等级**：1-10级
- **建造费用：100灵石（1级）
- **升级效果**：
  - 1级：修炼速度+50%
  - 3级：修炼速度+100%
  - 5级：修炼速度+150%，可时间加速×2
  - 7级：修炼速度+200%，可时间加速×3
  - 10级：修炼速度+300%，可时间加速×5
- **特殊功能**：
  - 闭关修炼（挂机）
  - 突破境界（提升成功率+10%）
  - 凝神静气（恢复神识）

#### 3. 灵田区
- **功能**：种植药材、灵植、灵果
- **初始**：2块灵田
- **扩展**：每升1级洞府+2块灵田
- **最大**：20块灵田（10级洞府）
- **建造费用**：5000灵石/块
- **升级效果**：
  - 1级：普通灵田，生长速度×1
  - 3级：上等灵田，生长速度×2
  - 5级：极品灵田，生长速度×3
  - 7级：仙品灵田，生长速度×5
  - 10级：神品灵田，生长速度×10，品质+1级
- **特殊功能**：
  - 种植药材（炼丹材料）
  - 种植灵果（食物、喂养灵兽）
  - 种植观赏植物（装饰、提升美观度）

## 五、洞府管理系统

### 1. 建筑布局

#### 自由建造模式
- 玩家可自由拖拽建筑位置
- 建筑可旋转方向（东南西北）
- 建筑之间需要间隔1米以上
- 某些建筑有朝向要求（如大殿要坐北朝南）

#### 一键布局
- 系统提供预设布局方案
- 可快速应用（适合新手）
- 包含：均衡布局、生产布局、战斗布局、美观布局

#### 布局保存
- 可保存当前布局
- 可加载之前的布局
- 最多保存5个布局方案

### 2. 资源管理

#### 灵石消耗
- 升级洞府：大量灵石
- 建造建筑：中等灵石
- 维护费用：每日自动扣除（根据洞府等级）
  - 1-3级：无维护费
  - 4-5级：每日100灵石
  - 6-7级：每日500灵石
  - 8-9级：每日10灵石
  - 10级：每日100灵石

#### 材料消耗
- 建造建筑需要特定材料
- 升级建筑需要更高级材料
- 材料可从秘境、采集、购买获得

### 3. 洞府防御

#### 防御值系统
- 洞府具有防御值（HP）
- 受到攻击时减少防御值
- 防御值归零则洞府受损

#### 防御手段
**被动防御**：
- 建筑本身耐久度
- 地形优势（山洞、悬崖等）
- 隐匿手段（幻阵）

**主动防御**：
- 布置防御阵法（自动反击）
- 布置杀阵（攻击入侵者）
- 灵兽守护（灵兽自动防御）
- 傀儡守卫（放置战斗傀儡）

#### 入侵机制
- NPC势力可能攻打洞府（剧情）
- 玩家可设置是否允许PVP入侵
- 洞府被攻破会损失部分资源

## 六、洞府特殊功能

### 1. 聚灵阵系统

#### 聚灵阵等级
| 等级 | 灵气加成 | 覆盖范围 | 建造费用 |
|------|---------|---------|---------|
| 一级聚灵阵 | +20% | 50㎡ | 5000灵石 |
| 二级聚灵阵 | +40% | 100㎡ | 5000灵石 |
| 三级聚灵阵 | +80% | 200㎡ | 1万灵石 |
| 四级聚灵阵 | +150% | 500㎡ | 5万灵石 |
| 五级聚灵阵 | +300% | 1000㎡ | 10万灵石 |
| 六级聚灵阵 | +500% | 全洞府 | 50万灵石 |

#### 聚灵阵布置
- 需要在阵法区布置
- 消耗阵旗、阵盘、灵石
- 可叠加多个聚灵阵（效果叠加）
- 需要定期维护（每月补充灵石）

### 2. 空间扩展系统

#### 扩展方式
**使用空间扩展符**：
- 一次性道具，使用后永久扩展
- 每次扩展+100㎡
- 价格递增：第1次1万灵石，第2次20万，第3次40万...

**使用空间扩展阵**：
- 需要阵法师Lv.10
- 布置复杂阵法
- 可扩展×2空间（如1000㎡→2000㎡）
- 费用：10万灵石

**洞天开辟**（高阶）：
- 需要化神期
- 开辟独立小空间
- 可无限扩展
- 需要空间法宝作为载体

### 3. 时间流速调整

#### 时间加速
- 在时光塔中可时间加速
- 加速倍率：×2至×10
- 消耗：每小时100中品灵石/倍率
- 应用：修炼、种植、孵化等

#### 时间减缓（特殊）
- 保存物品，防止腐朽
- 冷冻时间，暂停成长
- 布置时间减缓阵

### 4. 洞府传送

#### 回城功能
- 使用**回城符**立即返回洞府
- 或使用洞府令牌（绑定）
- 冷却时间：10分钟

#### 对外传送
- 在洞府传送阵可传送到：
  - 门派
  - 各大城镇
  - 秘境入口
  - 好友洞府（需授权）

## 七、洞府与其他系统联动

### 1. 与炼丹系统
- 炼丹房提供专业炼丹场所
- 地火品质影响炼丹成功率
- 药材可在灵田种植
- 炼制的丹药存放在宝库

### 2. 与炼器系统
- 炼器室提供专业炼器场所
- 地火熔炉提供高温
- 炼制的法宝可在宝库展示

### 3. 与符箓系统
- 符道堂提供绘制场所
- 安静环境提升成功率
- 符箓可用于洞府防御

### 4. 与阵法系统
- 阵法区可布置各种阵法
- 聚灵阵、防御阵、杀阵等
- 阵法覆盖全洞府

### 5. 与灵宠系统
- 灵兽园可容纳灵宠
- 灵宠在此休息、繁殖
- 灵宠可守护洞府

### 6. 与食物系统
- 灵田种植食材
- 厨房烹饪灵食
- 食物存放在宝库

### 7. 与修炼系统
- 修炼密室提升修炼速度
- 灵气浓度影响修炼效率
- 突破境界成功率提升

### 8. 与货币系统
- 建造、升级消耗大量灵石
- 维护费用持续消耗
- 洞府是灵石消耗大户

### 9. 与门派系统
- 门派可赠送初始洞府
- 门派提供洞府折扣
- 某些洞府只有门派弟子可购买

### 10. 与社交系统
- 会客厅接待好友
- 好友可访问洞府
- 展示成就和收藏

## 八、洞府获取方式

### 1. 初始赠送
- 新手村赠送1级简陋洞府（100㎡）
- 位置：新手村附近山洞
- 免费，永久使用

### 2. 门派赠送
- 加入门派后赠送2级小型洞府
- 位置：门派后山
- 免费，但离开门派后收回

### 3. 购买
**凡间房产商**：
- 1-3级洞府（5万-1万灵石）
- 位置：凡间各处

**修仙坊市**：
- 4-5级洞府（50万-10万灵石）
- 位置：灵山、海岛等

**拍卖行**：
- 6级以上洞府（竞价）
- 位置：福地、洞天等稀有地点

### 4. 任务奖励
- 完成特定任务获得洞府
- 通常是剧情洞府，位置特殊
- 自带特殊效果

### 5. 自主开辟
- 金丹期后可自主开辟洞府
- 选择野外合适地点
- 消耗：5万灵石 + 开山符×10
- 优点：可选择最佳风水位置

### 6. 击败敌人夺取
- 击败敌对修士可夺取其洞府
- 需要破解洞府防御
- 风险高，收益大

## 九、特色玩法

### 1. 洞府评级系统
- 系统对洞府进行综合评分
- 评分维度：
  - 灵气浓度（30分）
  - 建筑数量和等级（20分）
  - 风水格局（20分）
  - 美观度（15分）
  - 防御力（15分）
- 总分：100分
- 排行榜：全服洞府评分排行

### 2. 洞府参观
- 开放洞府供其他玩家参观
- 参观者可点赞、评论
- 点赞数影响排名
- 洞府主人可获得"人气值"奖励

### 3. 洞府设计大赛
- **每季度举办**一次
- 玩家提交自己的洞府设计
- 其他玩家投票
- 奖励：
  - 第一名：传说建筑图纸、100万灵石、称号"洞天设计师"
  - 前十名：稀有建筑图纸、10万灵石

### 4. 洞府守卫战
- 定期刷新妖兽攻打洞府事件
- 玩家需要守卫洞府
- 成功守卫：获得奖励
- 失败：洞府受损，需要修复

### 5. 洞府联盟
- 多个玩家的洞府可组成联盟
- 共享防御、资源
- 联盟洞府可互相传送
- 组建洞府城市

## 十一、平衡性设计

### 1. 升级成本递增
- 越高级洞府升级费用指数增长
- 限制玩家快速冲级
- 需要长期积累

### 2. 维护费用
- 高级洞府需要持续投入维护
- 防止囤积高级洞府
- 形成灵石消耗循环

### 3. 境界限制
- 某些高级洞府需要对应境界
- 避免低境界玩家过于强大
- 境界和洞府同步成长

### 4. 建筑数量限制
- 不同等级洞府建筑数量有上限
- 需要合理规划布局
- 优先建造核心建筑
