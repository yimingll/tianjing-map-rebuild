# 修仙游戏装备系统设计方案 v2.0 - 世界观对齐版

> **"法宝者,修士之第二性命也。剑修以剑为命,丹修以炉为本,器修以锤为魂。"**
>
> 本文档基于《玄鉴仙录·卷五·市井烟火》《卷四·修行之道》,定义青玄界真实的法宝装备体系

## 一、法宝概述

### 1.1 何为法宝?

在青玄界,修士所用之器物,统称为"法宝"。

与凡间兵器不同,法宝需以灵力催动,方能发挥威能。

**白眉老者说书**:

> "诸位客官,今日老夫讲的,是'法宝'二字。"
>
> "何为法宝?凡人一把铁剑,斩树木如切菜,此为凡器。"
> "修士一柄飞剑,御剑飞行,千里之外取人首级,此为法宝。"
>
> "凡器靠力气,法宝靠灵力。"
> "练气期修士,一把下品法器,可斩金丹妖兽。"
> "金丹期修士,一件上品法宝,可平山断岳。"
>
> "修行之道,三分靠功法,七分靠法宝。"
> "同境界修士,法宝好者胜,法宝差者败。"
> "江湖有言:'工欲善其事,必先利其器'便是此意。"

### 1.2 法宝分类

**按用途分**:
- **武器类**: 飞剑、刀枪、法器
- **护具类**: 道袍、灵甲、护盾
- **辅助类**: 储物戒、灵兽袋、丹炉
- **特殊类**: 符箓、阵盘、傀儡

**按修士类型分**:
- **剑修**: 飞剑为主,一剑破万法
- **体修**: 灵甲为主,肉身成圣
- **法修**: 法袍为主,施法增强
- **丹修**: 丹炉为本,炼丹成道
- **器修**: 炼器炉为魂,铸器成就

## 二、法宝等阶体系

### 2.1 青玄界法宝等阶

> **重要**: 基于《玄鉴仙录·卷五·市井烟火》的炼器等级

| 等阶 | 名称 | 适用境界 | 价格参考 | 代表法宝 |
|------|------|---------|---------|---------|
| **0阶** | 凡器 | 凡人-练气初期 | 数两银子 | 凡铁剑、精钢刀 |
| **1阶** | 法器 | 练气期-筑基期 | 20-1000下品 | 青钢剑、灵木盾 |
| **2阶** | 上品法器 | 筑基期-金丹期 | 500上品 | 紫霄剑 |
| **3阶** | 法宝 | 金丹期-元婴期 | 数万上品 | 翻天印、七星剑 |
| **4阶** | 灵宝 | 元婴期-化神期 | 数十万上品 | 混元珠 |
| **5阶** | 仙器 | 化神期以上 | 无价之宝 | 玄天神剑 |

**真实案例**:
```
【玄天神剑】
玄天宗镇宗之宝,上古仙剑,传说可斩日月星辰。
现由掌门云无极掌管,威震天下三百年。
血河宗曾派元婴长老前来夺剑,被云无极一剑斩杀。

【干将莫邪双剑】
铸剑山庄镇庄之宝,神兵利器,削铁如泥。
欧冶子大师亲手所铸,耗时三十年,用尽百种灵材。
双剑合璧,威力倍增,可斩化神大能。

【七星剑阵】
传说中的上古剑阵,七把飞剑,威力惊天。
独孤求败曾在绝情谷底寻得其中三把。
江湖传闻,集齐七剑者,可纵横天下。
```

### 2.2 等阶详解

#### 0阶 - 凡器

凡人打造的兵器,无需灵力,纯靠力气。

**特点**:
- 凡人铁匠打造
- 材料为凡铁、精钢
- 不含灵气
- 练气期修士很快就会淘汰

**价格**: 几两银子到十两黄金
**代表**: 凡铁剑、精钢刀、铁锤

#### 1阶 - 法器

修士入门装备,需灵力催动。

**特点**:
- 含有少量灵气
- 可以用于修炼
- 普通炼器师即可打造
- 练气期修士的主力武器

**获取方式**:
- 白云坊市购买 (20-1000下品灵石)
- 门派兑换 (100-500贡献度)
- 自己炼制 (需炼器技能)
- 击杀妖兽掉落

**真实案例**:
```
【白云坊市的法器摊】
散修张三在白云坊市摆摊卖法器。
一把下品青钢剑,标价50下品灵石。
"客官您看,这剑含有金属性灵气,适合金灵根修士。"
"虽是下品,但用料扎实,斩练气十层妖兽绰绰有余!"

散修李四犹豫半天,还是买下了。
他攒了三个月的灵石,就为这把剑。
```

#### 2阶 - 上品法器

筑基期修士的主流装备,金丹期修士也会使用。

**特点**:
- 灵气充沛
- 威力巨大
- 需要筑基期以上修士才能发挥全部威力
- 高级炼器师打造

**获取方式**:
- 铸剑山庄定制 (500上品灵石起)
- 拍卖会竞拍
- 秘境遗迹获得
- 门派赏赐

**真实案例**:
```
【铸剑山庄定制】
金丹修士王大力,慕名前往铸剑山庄。
欧冶子大师亲自接待:"您要定制什么法器?"
"一把重剑,越重越好,我是体修。"

欧冶子点头:"重剑无锋,大巧不工。三个月后来取。"
"价格...1000上品灵石。"

王大力咬牙:"成交!"
三个月后,王大力拿到重剑,重达千斤。
一剑斩出,山峰崩塌。

王大力大喜:"好剑!不愧是铸剑山庄!"
```

#### 3阶 - 法宝

金丹期以上修士的法宝,威力惊人。

**特点**:
- 威力巨大,可开山裂石
- 极品法宝可能诞生器灵
- 价值连城,一般修士买不起
- 只有顶级炼器师才能打造

**获取方式**:
- 上古遗迹获得
- 师门传承
- 铸剑山庄顶级订制 (数万上品灵石)
- 机缘巧合

**真实案例**:
```
【独孤求败的重剑】
独孤求败年少时,在绝情谷底发现一把玄铁重剑。
重剑无锋,黑如墨,重如山。
独孤求败以此剑纵横江湖五十年,未尝一败。

江湖人称:"独孤剑,天下无敌。"
后来独孤求败自言:"重剑在手,世间再无对手。"
于是将重剑沉入谷底,改用木剑。

【血河宗的血海幡】
血河宗镇宗法宝,用十万亡魂炼制。
幡一展开,血海滔天,鬼哭神号。
云无极曾与血无涯大战,一剑斩断血海幡一角。
血无涯重伤,血河宗从此不敢小觑玄天宗。
```

#### 4阶 - 灵宝

元婴期修士的本命法宝,威力通天。

**特点**:
- 必有器灵
- 可随主人境界成长
- 损坏则主人受伤
- 世间罕有

**获取方式**:
- 元婴期自己炼制
- 机缘获得
- 前辈赠予

**真实案例**:
```
【云无极的凌霄剑】
云无极除了玄天神剑,还有一把本命飞剑"凌霄"。
此剑乃云无极筑基时所炼,陪伴他两百余年。
如今云无极化神,凌霄剑也晋升为灵宝。

剑中器灵名叫"小霄",性格活泼。
云无极与小霄心意相通,战斗时配合无间。

江湖人都知道,云无极的本命剑凌霄,才是他的最强底牌。
```

#### 5阶 - 仙器

传说中的神兵,威力无穷。

**特点**:
- 上古仙人遗留
- 威力惊天动地
- 青玄界罕有
- 无价之宝

**代表**:
- 玄天神剑 (玄天宗)
- 干将莫邪双剑 (铸剑山庄)
- 血河大阵 (血河宗)

## 三、法宝获取途径

### 3.1 购买

#### 白云坊市

中州最大的修士交易市场。

**可购买法宝**:
- 下品法器: 20-100下品灵石
- 中品法器: 100-500下品灵石
- 上品法器: 500-2000下品灵石

**注意事项**:
- 坊市中鱼龙混杂,十件九假
- 需要眼力,否则容易被骗
- 有店铺的商家相对可靠
- 散修摆摊便宜但风险高

**真实案例**:
```
【买到假货】
散修王铁匠在白云坊市买到一把"上品法剑"。
卖家说:"此剑含有雷属性,威力巨大!"
王铁匠花了500下品灵石买下。

结果用了三天,剑就碎了。
原来是下品法器用符箓伪装的。
王铁匠气得吐血,但卖家早已跑路。

【火眼金睛】
老修士李药师在白云坊市淘宝。
他看中一个破碗,摊主要价10下品灵石。
李药师一眼看出,这是上古炼丹炉的残片!

他装作不在意,讨价还价:"5灵石卖不卖?"
摊主:"成交!"

李药师回去一研究,发现这残片可以提升炼丹成功率!
他赚大了!
```

#### 铸剑山庄

青玄界最好的炼器圣地。

**特点**:
- 品质保证,童叟无欺
- 价格昂贵,但物有所值
- 可以定制
- 需要排队,顶级大师要等数年

**定制流程**:
1. 前往铸剑山庄
2. 说明需求
3. 大师评估材料和时间
4. 缴纳定金 (50%)
5. 等待打造
6. 验收付款

**价格参考**:
- 普通炼器师: 500上品灵石起
- 高级炼器师: 2000上品灵石起
- 欧冶子大师: 10000上品灵石起 (不一定接单)

**真实案例**:
```
【欧冶子的脾气】
某金丹修士傲慢无礼,对欧冶子颐指气使。
"给我炼把剑,要最好的!"

欧冶子当场翻脸:"滚!我铸剑山庄不接你的单!"

金丹修士恼羞成怒,要动手。
铸剑山庄数十位弟子围了上来,个个都是炼器高手。

金丹修士悻悻离去。

此事传开,江湖人都说:"铸剑山庄的人,惹不起。"
```

各大宗门都有本门特色法宝,可用贡献度兑换。

### 3.3 击杀妖兽

妖兽掉落材料,可以炼制法宝。

**掉落规律**:
- 练气期妖兽: 掉落凡铁、兽皮
- 筑基期妖兽: 掉落灵材、兽骨
- 金丹期妖兽: 掉落天材地宝

### 3.4 遗迹机缘

上古遗迹中,常有前辈遗留的法宝。

**真实案例**:
```

独孤求败年少时,误入绝情谷底。
谷底有一前辈洞府,遗留三把飞剑。

他以此三剑,修炼剑道,终成一代剑圣。
```

学习炼器技能,可以自己炼制法宝。

**炼器流程**:

2. 画设计图
3. 炼器炉中炼制
4. 加入灵气
5. 成器or失败

**成功率**:
- 下品法器: 70%
- 中品法器: 50%
- 上品法器: 30%
- 法宝: <10%

## 四、法宝强化与进阶

### 4.1 强化

使用灵石和材料,可以强化法宝。

**强化效果**:
- 提升威力
- 增加耐久
- 改善手感

**强化风险**:
- 有失败概率
- 失败损失材料
- 极少数情况下法宝损坏

### 4.2 炼化

修士获得法宝后,需要炼化才能使用。

**炼化过程**:
- 将法宝与自身灵力融合
- 建立心神联系
- 如臂使指

**炼化层数**:
- 第一层: 基础使用
- 第三层: 熟练使用
- 第五层: 人剑合一
- 第七层: 心念相通
- 第九层: 器主不分

**真实案例**:
```
【云无极的剑道】
云无极年少时,获得一把飞剑。
他苦练十年,将飞剑炼化至九层。

此剑与他心意相通,战斗时剑随心动。
敌人根本无法预判剑的轨迹。

云无极凭此剑,纵横江湖三百年。
```

### 4.3 进阶

极品法宝,可能进阶为灵宝。

**进阶条件**:
- 法宝强化至极限
- 主人境界突破
- 大量灵气灌注
- 机缘巧合

## 五、器灵系统

### 5.1 何为器灵?

法宝有灵,谓之器灵。

**白眉老者说书**:

> "诸位可知,法宝用久了,会生出灵性?"
>
> "这灵性,便是'器灵'。"
>
> "器灵者,法宝之魂也。"
>
> "有器灵的法宝,威力倍增,可独立战斗。"
>
> "但器灵也有脾气,若是性格不合,反而碍事。"
>
> "江湖传闻,云无极的本命剑凌霄,器灵名叫小霄。"
> "此灵活泼可爱,但战斗时凶猛无比。"
> "云无极与小霄配合,如虎添翼。"

### 5.2 器灵诞生

**诞生条件**:
- 法宝品质极高 (极品以上)
- 使用时间极长 (百年以上)
- 灵气滋养充沛
- 机缘巧合

**诞生方式**:

**自然诞生**:
长时间使用,法宝自然产生灵性。
这种器灵品质最高,忠诚度最高。

**主动孕育**:
使用妖兽魂魄+灵气灌注,强行催生器灵。
这种器灵可能有妖兽习性,需要驯服。

**高人点化**:
找化神期以上大能,用神识点化。
这种器灵100%成功,但品质看大能实力。

### 5.3 器灵性格

器灵也有性格:
- **忠诚型**: 绝对服从主人,最可靠
- **傲娇型**: 偶尔闹脾气,但关键时刻靠谱
- **活泼型**: 话多,但能力强
- **沉默型**: 不说话,默默守护
- **叛逆型**: 需要培养忠诚度,否则可能反噬

**真实案例**:
```
【器灵反噬】
魔修血屠夫,强行用活人魂魄炼制器灵。
器灵充满怨气,表面服从,暗中记仇。

某次战斗,血屠夫危急时,器灵突然反噬!
血屠夫被自己的法宝斩杀,死状凄惨。

江湖人都说:"器灵不可虐,否则必反噬。"
```

### 5.4 器灵能力

**被动能力**:
- 增强法宝威力
- 自动修复法宝
- 灵力恢复加速

**主动能力**:
- 器灵化形,协助战斗
- 器灵警示,提前预警
- 器灵共鸣,增幅攻击

## 六、本命法宝

### 6.1 何为本命法宝?

修士亲手炼制,或认主的法宝,称为"本命法宝"。

**特点**:
- 与修士性命相连
- 法宝强,修士强
- 法宝损,修士伤
- 法宝毁,修士境界跌落

**优势**:
- 威力提升50%以上
- 心念相通,指挥如臂
- 可随境界成长
- 损坏可用修为修复

**劣势**:
- 每人只能有1件本命法宝
- 法宝被毁,代价巨大

**真实案例**:
```
【本命法宝被毁】
血河宗长老血屠夫,本命法宝是血海幡。
与云无极大战时,血海幡被一剑斩断。

血屠夫当场重伤,境界从元婴中期跌落至元婴初期。
他吐血三升,修为大损。

从此血屠夫闭关十年,才恢复伤势。
但境界再也无法突破。
```

## 八、法宝耐久与修复

### 8.1 耐久度

法宝使用久了,会损耗。

**损耗原因**:
- 战斗使用
- 灵气耗尽
- 受到破坏性攻击
- 主人死亡

**耐久度归零**:
- 法宝无法使用
- 需要修复
- 修复不及时会彻底损坏

### 8.2 修复方式

**NPC修理**:
- 找铸剑山庄修理
- 消耗灵石
- 修复速度快

**自己修复**:
- 需要炼器技能
- 消耗材料
- 成本低但需要时间

**真实案例**:
```
【修复失败】
散修李铁匠的飞剑损坏,他舍不得花钱修。
自己修了三次,都失败了。
最后飞剑彻底损坏,成了废铁。

李铁匠后悔莫及:"早知道就花钱去铸剑山庄了!"
```
