# 修仙游戏属性系统完整方案

> 基于主流修仙游戏（一念逍遥、想不想修真、凡人修仙传）和网络小说经典设定
>
> **重要**: 本文档为属性系统唯一数据源,其他系统应引用本文档的属性计算公式
>
> **依赖系统**: 境界突破系统(境界定义)、货币系统(灵石价格)
>
> **参考标准**: 详见《【重要】文档统一标准.md》

---

## 一、核心资源属性

### 1. 气血/生命值（HP）
- **作用**：体修的核心属性，相当于生命总值
- **计算公式**：`基础气血 = 体质 × 1 + 境界加成 + 功法加成`
- **特点**：气血为0进入重伤状态，需要丹药或休息恢复

### 2. 真元/灵力（MP）
- **作用**：法修的核心属性，施展法术消耗，也是法修的"第二层血条"（法术护盾）
- **计算公式**：`真元 = 灵根 × 1 + 境界加成 + 功法加成`
- **恢复方式**：自然恢复、打坐、灵石、丹药

### 3. 神识/元神
- **作用**：影响感知范围、法术威力、炼丹炼器成功率
- **计算公式**：`神识强度 = 境界 × 悟性系数`
- **特殊能力**：高境界可以用神识攻击（元神攻击）

## 二、基础属性（六维）

### 1. 体质 - 生命之基
- **影响**：气血上限、物理防御、异常状态抗性、生命恢复速度
- **成长**：体修加成最高，每点 +1.5气血

### 2. 力量 - 物理之力
- **影响**：物理攻击、负重、击退效果
- **成长**：剑修/体修侧重，每点 +0.2物理攻击

### 3. 敏捷 - 身法速度
- **影响**：闪避率、暴击率、先攻值、技能冷却缩减
- **成长**：剑修侧重，每点 +0.5%闪避

### 4. 灵根 - 修仙资质
- **影响**：真元上限、法术威力、修炼速度
- **成长**：法修加成最高，每点 +1.2真元

### 5. 悟性 - 领悟之道
- **影响**：技能领悟速度、突破成功率、顿悟几率、神识强度
- **成长**：所有流派都需要，但提升最难

### 6. 气运 - 天命所归
- **影响**：暴击率、掉落率、遇到奇遇概率、天劫难度（负向影响）
- **特殊**：隐藏属性，难以直接提升

## 四、境界系统（详细划分）

## 五、战斗属性计算公式

### 伤害计算

#### 物理伤害
```
基础物理伤害 = 攻击力 × 攻击力 / (攻击力 + 防御力 × 0.6)
实际伤害 = 基础伤害 × (1 + 暴击加成) × (1 - 减伤率) × 属性克制系数
```

#### 法术伤害
```
基础法术伤害 = 法术攻击 × (1 + 灵根加成) - 法术防御 × 0.5
实际伤害 = 基础伤害 × (1 + 法术强度%) × (1 - 法术抗性) × 法则加成
```

### 防御计算

#### 减伤率
```
物理减伤 = 防御力 / (防御力 + 999)
法术减伤 = 法抗 / (法抗 + 999)
```

#### 实际减伤
```
最终减伤 = 基础减伤 × (1 - 穿透率) + 护盾减伤 + 境界压制
```

### 二级属性详细

#### 攻击类
- **物理攻击** = 力量 × 0.2 + 武器 + 境界加成
- **法术攻击** = 灵根 × 0.25 + 法宝 + 功法加成
- **暴击率** = 5% + 敏捷/20 + 装备加成
- **暴击伤害** = 150% + 技能/装备加成
- **穿透** = 武器属性 + 境界压制

#### 防御类
- **物理防御** = 体质 × 0.15 + 护甲 + 体修功法加成
- **法术防御** = 灵根 × 0.12 + 法宝 + 护体神通
- **闪避率** = 敏捷/15 + 身法加成
- **格挡** = 体质/10（体修专属）

#### 速度类
- **攻击速度** = 100% + 敏捷/20
- **施法速度** = 100% + 悟性/30
- **移动速度** = 基础值 + 敏捷/10 + 身法

## 六、修炼方向/职业系统（动态倾向系统）

> **设计理念**：不再预设固定职业，而是根据玩家属性分配动态判定修炼倾向，参考《流放之路》天赋树系统

### 核心机制

#### 1. 倾向计算公式

根据玩家六维属性的分配比例，实时计算五大修炼倾向：

```
体修倾向 = 体质 × 2.0 + 力量 × 1.0
剑修倾向 = 力量 × 1.5 + 敏捷 × 1.5 + 悟性 × 0.5
法修倾向 = 灵根 × 2.0 + 悟性 × 1.0
鬼修倾向 = 悟性 × 2.0 + 气运 × 1.0 + 灵根 × 0.5
器修倾向 = 悟性 × 2.0 + 体质 × 0.8 + 灵根 × 0.8

倾向百分比 = 各倾向值 / 总倾向值 × 100%
```

#### 2. 修炼路线判定

**【纯修路线】**
- **条件**：某个倾向 ≥ 40%
- **效果**：该流派技能威力 +30%，属性效果 +20%
- **称号**：体修宗师、剑道大成、法修真君等
- **特点**：专精一道，威力最强

**【双修路线】**
- **条件**：两个倾向都 ≥ 25%，其他 < 20%
- **效果**：两个流派技能威力各 +15%，属性效果各 +10%
- **称号**：剑体双修、法体兼修等
- **特点**：兼顾两道，灵活多变

**【杂修路线】**
- **条件**：没有倾向 ≥ 25%
- **效果**：所有技能都可学，但无额外加成
- **称号**：杂家、万法归一
- **特点**：全能发展，前期弱后期强

### 属性重置机制

**【初级洗点符】**
- 消耗：100灵石
- 效果：重置所有属性点
- 限制：每个大境界只能用1次
- 获取：商店购买、任务奖励

**【高级洗点丹】**
- 消耗：筑基丹×3 + 1000灵石
- 效果：完全重置 + 保留80%修炼经验
- 限制：1个月冷却
- 获取：炼丹、高级商店

**【逆天改命】**
- 消耗：天材地宝 + 渡一次天劫
- 效果：完全重置 + 可改变灵根
- 风险：失败会掉1个小境界
- 获取：特殊剧情触发

## 七、特色系统

### 1. 天劫系统

#### 触发条件
- 境界突破
- 逆天改命
- 杀戮过重

#### 劫难类型
- **雷劫**：最常见，物理+法术混合伤害
- **心魔劫**：考验道心，悟性检定
- **五行劫**：五种元素轮番攻击
- **人劫**：人为干扰，需应对敌人

#### 天劫强度计算
```
基础强度 = 境界难度系数
实际强度 = 基础 × (1 + 气运惩罚) × (1 + 因果值/100)
```

---

### 2. 顿悟系统

#### 触发条件
- 随机触发（悟性/100 的概率）
- 特定场景触发（观察天地异象）
- 战斗中触发（生死边缘）

#### 顿悟收益
- 经验翻倍（持续时间：1小时）
- 技能熟练度大增
- 突破成功率 +20%
- 可能领悟新技能/法则

---

## 九、数值平衡建议

### 软上限机制
- 单项属性超过 **200** 后，每+1的收益递减 **20%**
- 超过 **500** 后，递减 **50%**
- 鼓励玩家均衡发展

### 境界压制
- **高1个大境界**：全属性+20%，命中+15%，闪避+10%
- **高2个大境界**：全属性+50%，受到伤害-30%
- **高3个大境界**：基本无法战胜（除非装备/技能碾压）

### 属性转化效率
```
1点体质 = 1.5气血 = 0.15物防 = 0.3抗性

1点敏捷 = 0.5%闪避 = 0.3%暴击 = 1攻速
1点灵根 = 1.2真元 = 0.25法攻 = 1%修炼速度
1点悟性 = 1%突破率 = 0.5神识 = 0.1%顿悟率
```

## 十一、属性提升途径

### 1. 境界突破
- 每次突破获得固定属性点分配
- 大境界突破属性翻倍成长

### 2. 装备加成

> **装备品阶系统**: 详见《【重要】文档统一标准.md》和《修仙游戏装备系统设计方案.md》

- **武器**：主要加攻击力、暴击
- **防具**：加防御、气血
- **法宝**：加法术攻击、真元
- **饰品**：加特殊属性（暴击伤害、穿透等）

装备品阶：0阶凡器 → 1阶法器 → 2阶灵器 → 3阶宝器 → 4阶灵宝 → 5阶通天灵宝 → 6-8阶仙器 → 9阶先天灵宝

### 3. 丹药提升
- **临时丹药**：增益持续时间有限（如爆灵丹+50%攻击，持续1小时）
- **永久丹药**：直接提升基础属性（如洗髓丹+0.5体质）

### 4. 功法被动
- 修炼高级功法获得永久属性加成
- 功法等级越高，被动效果越强

### 5. 称号加成
- 完成特殊成就获得称号
- 例如：「百人斩」+1力量，「丹道宗师」+1.5悟性

### 6. 洞府/环境
- 在灵气浓郁的地方修炼有修炼速度加成
- 特殊地形可能有属性临时加成

### 7. 天材地宝
- 服用珍稀材料永久提升属性
- 例如：千年人参+1体质，龙血果+0.5力量

## 十二、进阶系统建议

### 1. 属性觉醒
- 某项属性达到特定阈值，解锁特殊能力
- 例如：体质达到100，觉醒「不灭金身」（数值保持不变，作为里程碑）

### 2. 体质特化
- 选择修炼方向（体修、剑修、法修等）
- 特化后获得该方向的专属技能树

### 3. 双修系统
- 同时修炼多种功法
- 优点：技能多样化
- 缺点：降低专精效率

### 4. 逆天改命
- 消耗大量资源，重置灵根或属性
- 代价：引发人劫/天劫
