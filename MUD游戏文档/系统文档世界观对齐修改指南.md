# 系统文档世界观对齐修改指南

> **目的**: 为所有游戏系统设计文档提供统一的世界观对齐修改指导
>
> **适用范围**: 所有《修仙游戏XXX系统设计方案.md》文档
>
> **参考文档**: 《玄鉴仙录》世界观典籍(17个背景文档)

---

## 一、修改原则

### 1.1 叙事风格转变

**从**: 游戏说明书风格
```
### 天剑宗
- 门派加成: 物理攻击 +30%, 暴击率 +15%
- 特色: 剑道第一
```

**改为**: 说书人叙事风格
```
### 玄天宗 ⚔️ - 剑修圣地

位于中州昆仑山脉主峰,掌门凌霄剑尊·云无极(化神后期)。

玄天宗弟子主修剑道,辅以雷法。门中剑气纵横,弟子出门在外,
一柄飞剑便是身份象征。但宗门内部派系斗争激烈...
```

### 1.2 去数值化原则

**避免**: 纯数值描述
- ❌ "+30%攻击力"
- ❌ "成功率70%"
- ❌ "伤害公式: ATK × 1.5"

**改为**: 文学化描述
- ✅ "剑道第一,飞剑术天下无双"
- ✅ "十人中有七人能够成功"
- ✅ "一剑之威,足以开山裂石"

**注意**: 游戏机制必要的数值可保留,但要配合叙事说明

### 1.3 世界观对齐要点

| 系统 | 必须对齐的内容 | 参考文档 |
|------|-------------|---------|
| 门派系统 | 门派名称、NPC名字、地理位置 | 卷三·江湖风云、宗门详录 |
| 境界系统 | 境界名称、突破过程、天劫描述 | 卷四·修行之道 |
| 货币系统 | 灵石价格、物价体系 | 卷五·市井烟火 |
| 装备系统 | 法宝名称、品质划分 | 卷三·江湖风云 |
| 炼丹系统 | 丹药名称、炼丹流派 | 卷二·诸子百家、宗门详录 |
| NPC系统 | 真实NPC名字、性格设定 | 人物志·江湖百家 |
| 地理系统 | 地名、城市、山川 | 卷一·天下形胜 |

## 二、各系统修改要点

### 2.1 门派系统

**必改内容**:
1. 门派名称对齐
   - ❌ 天剑宗、太虚宫、百花谷 → ✅ 玄天宗、太上道宗、灵药谷
2. 添加真实NPC
   - 掌门: 云无极、葛青云、血无涯等
3. 地理位置具体化
   - "某山" → "中州昆仑山脉主峰"
4. 组织架构对齐
   - 参考《宗门详录·组织架构》的真实结构

**示例**: 已完成 `修仙游戏门派系统设计方案_v2.0_世界观对齐版.md`

---

### 2.2 境界突破/渡劫系统

**必改内容**:
1. 境界体系完全对齐
   - 练气→筑基→金丹→元婴→化神→飞升
   - 成功率: "筑基成功率30%"而非"70%"
2. 真实的淘汰率
   - "十万人踏入修行,能筑基者不过千人"
3. 心魔劫描述
   - 执念心魔、恐惧心魔、欲望心魔
4. 天劫威力对齐
   - 化婴雷劫27道、化神雷劫81道
5. 散修vs宗门差距
   - 强调资源差距、成功率差距

**示例**: 已完成 `修仙游戏境界突破渡劫系统设计方案_v2.0_世界观对齐版.md`

---

### 2.3 属性系统

**必改内容**:
1. 灵根类型对齐
   - 五行灵根、变异灵根、特殊体质
2. 修炼方向去职业化
   - 不要"战士、法师、刺客"等游戏职业名
   - 改为"体修、剑修、法修、丹修"等修炼倾向
3. 属性影响文学化
   - "悟性影响突破成功率"而非"+10%成功率"

**参考**:
- 《卷四·修行之道》第五章:百家修法
- 《卷四·修行之道》第七章:资质与机缘

---

### 2.4 货币/交易系统

**必改内容**:
1. 灵石体系对齐
   - 下品、中品、上品、极品灵石
   - 兑换比例: 1:100
2. 物价参考真实世界观
   - 筑基丹: 1000上品灵石(5000-50000下品)
   - 客栈一晚: 1下品灵石
3. 交易场所具体化
   - "白云坊市"(中州)、"醉仙居"(东海)
4. 商帮名称对齐
   - 晋商、徽商、粤商

**参考**:
- 《卷五·市井烟火》第二章:坊市交易
- 《卷三·江湖风云》第四章:商帮镖局

---

### 2.5 装备/法宝系统

**必改内容**:
1. 品质划分对齐
   - 凡品→灵品→法宝→仙器
   - 每级细分: 下品、中品、上品、极品
2. 法宝命名风格
   - 避免"+10攻击的铁剑"
   - 改为"玄天神剑(上古仙剑,可斩日月星辰)"
3. 炼器宗门对齐
   - "铸剑山庄"(庄主:欧冶子)

**参考**:
- 《卷三·江湖风云》各宗门的镇宗之宝

---

### 2.6 炼丹系统

**必改内容**:
1. 丹药名称对齐
   - 筑基丹、化婴丹、破劫丹、续命丹
2. 炼丹流派
   - 灵药谷(谷主:葛青云)为炼丹圣地
3. 丹药品质
   - 下品、中品、上品、极品
4. 炼丹过程去游戏化
   - 不要"点击炼制按钮"
   - 改为"控火诀、凝丹诀、成丹"等步骤

**参考**:
- 《宗门详录·组织架构》灵药谷部分
- 《卷二·诸子百家》百艺流传

### 2.7 炼器系统

2. 炼器等级: 凡器→灵器→法宝→仙器
3. 炼器材料真实化
