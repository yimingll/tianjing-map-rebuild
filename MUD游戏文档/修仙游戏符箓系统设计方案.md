# 修仙游戏符箓系统设计方案
> **注**: 本方案为纯单机/买断制游戏设计，无充值系统

## 一、系统总览

### 核心设计理念
- **九品符箓体系**：一品至九品，威力递增
- **符纸/符墨品质**：影响符箓效果和成功率
- **符师职业路线**：专精符道可成为符道宗师
- **一次性消耗品**：符箓使用后消失，需持续制作
- **快捷战斗辅助**：战斗中快速释放，无需吟唱
- **符阵组合**：多张符箓可组成符阵，威力倍增

>
> **重要**: 本文档定义符箓系统
>
> **依赖系统**: 采集系统、境界系统
>
> **参考标准**: 详见《【重要】文档统一标准.md》

## 六、符箓绘制流程

### 绘制准备
1. **选择符箓类型** - 从符典中选择要绘制的符箓
2. **准备材料** - 符纸、符墨、符笔
3. **检查符师等级** - 确认可绘制该品阶
4. **选择绘制地点** - 安静环境可提升成功率

### 绘制过程（手动/自动）

#### 手动绘制模式（推荐）
**第一阶段：调墨（5秒）**
- 操作：调配符墨（保持浓度在绿色区域）
- 成功：+5%成功率
- 失败：-5%成功率

**第二阶段：起笔（3秒）**
- 操作：选择正确起笔位置（点击符纸对应位置）
- 成功：+5%成功率
- 失败：-10%成功率

**第三阶段：描绘（20秒）**
- 操作：沿着符文路径描绘（拖动鼠标/手指沿路径）
- 准确度越高，成功率和品质越高
- 成功：+10%成功率，+10%品质
- 失败：-10%成功率

**第四阶段：注灵（5秒）**
- 操作：注入灵力（按住蓄力，松开时停在绿区）
- 成功：+5%成功率，+5%品质
- 失败：符箓报废

**第五阶段：封印（自动）**
- 根据前四阶段表现，计算最终成功率和品质
- 成功则获得符箓，失败则材料消耗

#### 自动绘制模式
- 消耗符箓令牌（每日赠送20个）
- 按基础成功率自动绘制
- 无法获得手动操作加成
- 适合批量绘制低品符箓

### 绘制成功率计算
```
最终成功率 = 基础成功率 + 符师等级加成 + 符纸加成 + 符笔加成 + 手动操作加成 + 环境加成
```

基础成功率：
- 一品符：80%
- 二品符：70%
- 三品符：60%
- 四品符：50%
- 五品符：40%
- 六品符：30%
- 七品符：20%
- 八品符：10%
- 九品符：5%

## 七、符阵系统

### 符阵概念
- 将多张符箓按特定方位布置，形成符阵
- 符阵威力远超单张符箓
- 符阵可持续存在，范围作用

【三才冰阵】: 寒冰符×3 → 冰封领域（冰封范围内所有敌人）
【三才雷阵】: 雷击符×3 → 雷云风暴（持续降雷30秒）

## 八、符箓与其他系统联动

### 1. 与战斗系统

### 3. 与阵法系统

- 高级阵法可用符箓作为阵基

### 5. 与经济系统

- 高品质符箓可拍卖
- 接受符箓委托赚取收入

### 6. 与探索系统
- 使用符箓解决探索难题（破禁符开宝箱、寻宝符找宝物）
- 符箓是探险必备物资

## 九、特色玩法

### 1. 符道大会
- **每月举办**的符师竞赛
- 比拼速度绘制、品质绘制、符阵布置
- 奖励：传说符典、极品符笔、称号"当月符圣"

### 2. 符箓图鉴
- 收集系统：绘制过的所有符箓记录
- 完成度奖励：
  - 25%：符道经验+10%
  - 50%：绘制成功率+5%
  - 75%：解锁隐藏符箓×5
  - 100%：称号"符道大师"，全符箓效果+20%

### 3. 自创符箓
- 选择符箓类型（攻击/防御/辅助等）
- 设计符文路径
- 实验绘制（可能失败）
- 成功后记录为专属符箓

### 4. 符傀儡系统
- 使用符箓制作傀儡
- 符傀儡可自动战斗
- 符傀儡有耐久度，用完消失

### 5. 符阵守护
- 在洞府/家园布置符阵
- 符阵可自动防御入侵者
- 高级符阵可永久存在

## 十一、平衡性设计

### 1. 符箓消耗平衡
- 符箓是一次性消耗品，需持续制作
- 高品符箓材料昂贵，限制使用频率
- 战斗中可连续使用但消耗数量

### 2. 符师与其他职业平衡
- 符师可批量制作符箓出售，获得收益
- 符箓是战斗重要辅助，需求量大
- 但绘制消耗时间和材料

### 3. 符箓效果平衡
- 符箓威力不及同境界术法，但无需吟唱
- 符阵威力强大但布置耗时
- 高品符箓稀有且昂贵

### 4. 经济平衡
- 低品符箓便宜，高品符箓昂贵
- 符纸符墨持续消耗，形成经济循环
- 限制携带符箓数量（背包限制）
