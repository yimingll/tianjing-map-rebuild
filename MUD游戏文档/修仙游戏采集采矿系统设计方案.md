# 修仙游戏采集/采矿系统设计方案

> **注**: 本方案为纯单机/买断制游戏设计，无充值系统
>
> **重要**: 本文档为所有材料(草药/矿石)的唯一数据源,其他系统只能引用不能重新定义
>
> **依赖系统**: 境界系统(采集需求境界)、装备系统(采集工具品阶)、货币系统(材料价格)
>
> **被依赖系统**: 炼丹系统、炼器系统、符箓系统、食物系统(所有材料均从本系统引用)
>
> **参考标准**: 详见《【重要】文档统一标准.md》

## 一、系统总览

### 核心设计理念
- **材料唯一数据源**：本系统是所有职业系统（炼丹、炼器、符箓、食物）的材料来源
- **双职业路线**：采药师（药材采集）+ 矿工（矿石开采）
- **探索驱动**：寻找资源点驱动玩家探索世界
- **技能成长**：采集技能等级影响效率和稀有度
- **资源再生**：资源点定时刷新，形成循环
- **品质分级**：采集物品有品质和年份之分

9. **移植技术** - 可移植野生药材到洞府灵田

**高阶技能**：
10. **批量采集** - 一次采集周围5米内所有药材

13. **药王传承** - 可识别并采集传说药材

### 2. 药材采集机制

#### 采集流程
1. **发现药材**：在野外发现药材采集点
2. **鉴定药材**：查看药材种类、年份、品质
3. **判断难度**：采集难度显示（简单/普通/困难/极难）
4. **开始采集**：按住采集键（3-10秒）
5. **采集完成**：获得药材，采集点消失

#### 采集成功率计算
```
最终成功率 = 基础成功率 × (1 + 采药师等级加成) × (1 + 工具加成) × (1 + 境界加成) - 药材难度惩罚
```

**基础成功率**：
- 普通药材：90%
- 灵药：70%
- 珍稀灵药：50%
- 上古灵药：30%
- 天材地宝：10%

**难度惩罚**：
- 有守护妖兽：-20%
- 生长在险地：-15%
- 天气恶劣：-10%
- 夜晚采集：-5%

#### 采集工具

| 工具名称 | 采集速度 | 成功率加成 | 品质加成 | 获取方式 |
|---------|---------|-----------|---------|---------|
| 徒手 | ×1 | +0% | +0% | 初始 |
| 普通镰刀 | ×1.2 | +5% | +0% | 商店购买（10铜钱） |
| 灵木镰刀 | ×1.5 | +10% | +5% | 商店购买（1灵石） |
| 精钢镰刀 | ×2 | +15% | +10% | 炼器制作 |
| 宝器药锄 | ×3 | +20% | +15% | 炼器制作（法宝级） |
| 仙品药篮 | ×5 | +30% | +25% | 仙界获得 |

### 3. 药材年份系统

#### 年份影响
- **年份越高，药效越强**
- **年份越高，采集难度越大**
- **年份越高，守护妖兽越强**

#### 年份鉴定
- **外观判断**：颜色、大小、灵气波动
- **采药师技能**：品质鉴定技能可直接显示
- **鉴定符**：使用鉴定符一次性鉴定

#### 年份加成

|------|---------|---------|
| 1-10年 | +0% | ×1 |
| 10-50年 | +50% | ×5 |
| 50-100年 | +100% | ×10 |
| 100-500年 | +200% | ×50 |
| 500-1000年 | +500% | ×100 |

- **秘籍记载**：古籍中记载矿脉位置

- **个人开采**：单人慢速开采
- **招募矿工**：雇佣NPC矿工自动开采
- **灵兽挖矿**：某些灵兽可协助挖矿
- **傀儡挖矿**：制作挖矿傀儡自动开采
- **阵法采矿**：布置采矿大阵，自动开采

### 4. 矿石开采机制

#### 开采流程
1. **发现矿脉**：探测或随机遇到
2. **选择矿点**：进入矿脉，选择开采点
3. **开始挖掘**：使用镐子开采（5-30秒）
4. **获得矿石**：随机获得1-5个矿石
5. **矿脉耗尽**：开采一定次数后矿脉暂时消失

#### 开采工具

| 石镐 | ×1.3 | +5% | 100次 | 商店购买（20铜钱） |

## 五、采集与其他系统联动

### 1. 与炼丹系统
- **药材是炼丹主要材料**
- 采集的药材直接用于炼丹
- 高品质药材提升炼丹成功率
- 稀有药材炼制高阶丹药

### 2. 与炼器系统
- **矿石是炼器主要材料**
- 不同矿石炼制不同法宝
- 金属矿提纯后成为精铁、精铜等
- 稀有矿石炼制灵宝仙器

### 3. 与食物系统
- 采集灵果、灵菇等食材
- 某些药材可作为食材
- 灵米、灵麦等谷物采集

### 4. 与洞府系统
- 采集的药材可种植在灵田
- 矿石可存放在宝库
- 招募矿工在洞府附近开采

### 5. 与灵宠系统
- 某些灵兽可协助采集（灵鼠寻宝）
- 灵兽可守护矿脉
- 采集食材喂养灵兽

### 6. 与门派系统
- 门派任务要求采集特定材料
- 门派占据大型矿脉
- 贡献度可兑换采集工具

### 7. 与货币系统
- 采集物品可出售换取灵石
- 稀有材料价值高
- 灵石矿直接产出货币

### 8. 与战斗系统
- 采集高级资源需要击败守护妖兽
- 矿脉争夺战PVE/PVP
- 采集时可能遇到敌对修士

#### 刷新位置
- **固定刷新点**：特定位置固定刷新
- **随机刷新点**：随机在某片区域刷新
- **稀有刷新点**：小概率在隐藏地点刷新

### 2. 资源竞争

#### 野外采集竞争
- 多个玩家可同时看到资源
- 谁先采集谁获得
- 采集速度快的玩家占优

#### 矿脉占据
- 大型矿脉可被门派/公会占据
- 占据后可设置采矿权限
- 其他玩家需要权限才能开采
- 可发起矿脉争夺战

## 七、特色玩法

### 1. 采集大赛
- **每周举办**采集竞赛
- 比拼采集数量、稀有度、速度
- 奖励：
  - 第一名：传说工具、10万灵石、称号"采集大师"
  - 前十名：高级工具、1万灵石

### 2. 资源图鉴
- 收集系统：采集过的所有资源记录
- 完成度奖励：
  - 25%：采集经验+10%
  - 50%：采集速度+10%
  - 75%：解锁隐藏资源点×5
  - 100%：称号"资源通"，全采集+20%

### 3. 寻宝任务
- 接取寻找特定资源任务
- 根据线索寻找隐藏资源
- 完成后获得丰厚奖励

### 4. 矿脉战争
- 大型矿脉争夺战
- 门派/公会之间PVP
- 胜者占据矿脉，获得开采权

### 5. 资源交易
- 玩家间可交易资源
- 拍卖行拍卖稀有材料
- 开设资源店铺

## 九、平衡性设计

### 1. 采集时间成本
- 低级资源采集快（3秒）
- 高级资源采集慢（30秒）
- 防止快速刷资源

### 2. 资源稀有度
- 普通资源大量
- 稀有资源稀少
- 天材地宝极其罕见

### 3. 采集风险
- 高级资源有守护妖兽
- 险地采集有环境伤害
- 矿脉争夺有PVP风险

### 4. 工具耐久
- 工具有耐久度限制
- 需要定期修理或更换
- 形成工具消耗循环
