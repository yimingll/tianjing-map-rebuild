## 一、系统概述

### 0.11000000000000001 设计理念

**核心目标**:
- 背叛是有意义的选择,而非简单的善恶
- 背叛有深刻的后果和代价
- 不同类型的背叛有不同的动机和结局
- 玩家可以体验背叛者和被背叛者两种角度

**参考作品**:
- 《仙逆》(主角多次背叛,但都有理由)
- 《遮天》(各种派系争斗)
- 《三国演义》(吕布叛变,关羽义薄云天)
- 《庆余年》(陈萍萍的背叛)

### 0.12 背叛的类型

```python
背叛类型:
0.1. 叛师 - 背叛师父
0.2. 叛门 - 背叛门派
0.3. 叛国 - 背叛国家
0.4. 叛友 - 背叛盟友/结拜兄弟
0.5. 叛道 - 从正道堕入魔道
0.6. 叛爱 - 背叛道侣/爱人
```

## 二、叛师系统

### 0.21000000000000002 叛师的动机

#### 正当理由
```python
【师父邪恶】
- 师父虐待弟子
- 师父修炼邪功,屠杀无辜
- 师父准备用弟子做邪恶实验(夺舍/献祭)

示例剧情:
玩家发现: 师父在地下室囚禁修士,吸取精血
选择:
0.1. 揭发师父 → 叛师(正义)
0.2. 协助师父 → 堕魔
0.3. 逃离 → 被师父追杀

【理念冲突】
- 师父守旧,弟子创新
- 师父冷血,弟子重情
- 师父只求道,弟子想救世

示例剧情:
灭门惨案: 师父下令不许管闲事
玩家: 选择出手拯救 → 违抗师命 → 可能叛师

【利益诱惑】
- 敌对势力许以重利
- 更强大的师父愿意收徒
- 门派宝库诱惑

示例剧情:
魔道高手: "杀了你师父,我传你更强功法"
选择:
0.1. 拒绝 → 魔道声望-10
0.2. 接受 → 杀师,堕魔,获得魔功
```

#### 不义理由
```python
【贪婪】
- 觊觎师父宝物
- 想夺取掌门之位
- 嫉妒师兄弟

【野心】
- 师父限制玩家发展
- 玩家想自立门户
- 觉得师父太弱,碍事

【被蛊惑】
- 心魔作祟
- 被天魔蛊惑
- 被敌人设计
```

### 0.22000000000000003 叛师的后果

```python
【立即后果】
0.1. 师门追杀
   - 师父亲自追杀(高境界)
   - 师兄弟围剿
   - 持续时间: 永久(除非师父死亡)

0.2. 名声扫地
   - 声望 -500
   - 被标记为"叛徒"
   - 正道门派拒绝接纳

0.3. 师门技能受限
   - 已学功法可能失效(师父设禁制)
   - 无法再学师门武学
   - 可能被种下追踪印记

【长期后果】
0.1. 道心受损
   - 如果是不义叛师: 心魔 +5
   - 突破时心魔劫难度加倍
   - 可能走火入魔

0.2. 业力增加
   - 杀师: 业力 +100(极重)
   - 叛师: 业力 +30

0.3. 后人评价
   - 历史记载为"逆徒"
   - 后代弟子蒙羞
   - 难以建立新门派(谁敢拜师?)

【特殊后果】
- 师父临死诅咒
- 师门至宝反噬
- 天道惩罚(杀师者天劫增强)
```

### 0.22999999999999998 叛师的剧情路线

```python
【路线0.1: 大义灭亲】
背景: 师父堕魔,屠杀同门
玩家: 联合师兄弟讨伐师父
结局:
- 成功: 被推为新掌门,声望+300
- 失败: 全灭或逃亡

【路线0.2: 投靠魔道】
背景: 正道虚伪,魔道实诚
玩家: 杀师投魔
结局:
- 获得魔功,实力大增
- 成为正道公敌
- 道心崩塌,可能成为杀戮机器

【路线0.3: 自立门户】
背景: 师父太弱,限制发展
玩家: 离开师门,自创门派
结局:
- 艰难创业
- 师门不断骚扰
- 如果成功: 成为一代宗师

【路线0.4: 被迫叛师】
背景: 师父被夺舍,玩家被污蔑
玩家: 被迫逃亡,查明真相
结局:
- 揭露真相,为师父报仇
- 洗清冤屈,恢复声望

【路线0.5: 悔过自新】
背景: 一时冲动叛师,后悔莫及
玩家: 苦苦追寻师父,请求原谅
结局:
- 师父原谅: 重归师门,成为最忠诚弟子
- 师父不原谅: 终生悔恨,化悔恨为力量
- 师父已死: 痛不欲生,为师复仇/寻找复活之法
```

## 三、叛门系统

### 0.31 叛门与叛师的区别

```python
叛师: 针对个人(师父)
叛门: 针对组织(门派)

可能情况:
0.1. 叛师不叛门: 师父是叛徒,讨伐师父保卫门派
0.2. 叛门不叛师: 门派邪恶,师父正义,师徒一起叛门
0.3. 叛师又叛门: 师父和门派都抛弃
```

### 0.32 叛门的动机

```python
【门派腐败】
- 门派高层贪腐
- 欺压弟子
- 资源分配不公

示例: 内门弟子享尽荣华,外门弟子如猪狗

【理念不合】
- 门派堕落
- 门派太过保守
- 门派作恶

示例: 青云门突然改修魔功

【私人恩怨】
- 被门派陷害
- 爱人被门派所杀
- 家族被门派灭门

【利益诱惑】
- 敌对门派挖角
- 魔道许以重利
- 建立新门派
```

### 0.32999999999999996 叛门的后果

```python
【门派追杀】
- 追杀令: 奖励 1万灵石
- 持续时间: 直到玩家死亡或门派覆灭
- 追杀队伍: 1-5名门派精英

【江湖孤立】
- 友好门派断交
- 无法加入其他正道门派(你有前科)
- 只能投靠魔道或自立

【失去资源】
- 门派不再提供修炼资源
- 门派藏书阁的功法可能失效(限制)
- 门派人脉全部失去

【连坐】
- 如果有家族,家族被门派报复
- 朋友被牵连
- 师父被追责(教出叛徒)
```

## 六、叛道系统(堕魔)

### 0.61 正道vs魔道

```python
正道:
- 讲规矩、重情义、慢成长
- 功法温和,无副作用
- 突破稳健

魔道:
- 无规则、重实力、快成长
- 功法霸道,有副作用(如吸人精血)
- 突破快但风险高

中立:
- 既不正也不邪
- 只求自己强大
```

### 0.62 堕魔的触发

```python
【主动堕魔】
- 修炼魔功
- 吸收魔气
- 杀戮无辜

【被动堕魔】
- 被魔气侵蚀
- 走火入魔
- 心魔作祟

【选择堕魔】
- 为了强大力量
- 为了复仇
- 理念认同魔道

【意外堕魔】
- 夺舍
- 被种下魔种
- 误入魔道
```

### 0.63 堕魔的后果

```python
【外观变化】

- 魔气环绕

【性格变化】
- 嗜杀

【能力变化】
+ 攻击力 +5%
```

### 0.64 堕魔剧情路线


## 八、Evennia实现方案

### 0.8099999999999999 背叛系统核心

```python
# typeclasses/背叛Mixin.py
class 背叛Mixin:
    """背叛系统混入类"""

    def at_object_creation(self):
        super().at_object_creation()

        # 背叛记录
        self.db.背叛记录 = {
            "叛师": False,
            "叛门": False,
            "叛国": False,
            "叛友次数": 0,
            "堕魔": False,
            "叛爱次数": 0,
        }

        # 忠诚度
        self.db.忠诚度 = {
            "师父": 10,
            "门派": 10,
            "国家": 5,
            "道侣": 0,  # 没有道侣时为0
        }

        # 义气值
        self.db.义气值 = 5

        # 业力
        self.db.业力 = 0  # 0是中性,负数是功德,正数是罪孽

    def 背叛师父(self, 方式="叛逃"):
        """背叛师父"""
        if not hasattr(self.db, "师父") or not self.db.师父:
            self.msg("你没有师父")
            return False

        师父 = self.db.师父

        # 记录背叛
        self.db.背叛记录["叛师"] = True

        # 标记
        self.tags.add("叛师者", category="恶名")

        # 后果
        self.db.忠诚度["师父"] = 0
        self.db.声望 -= 500

        if 方式 == "杀师":
            self.db.业力 += 100  # 极重业力
            self.db.心魔 = self.db.get("心魔", 0) + 5
            self.tags.add("弑师者", category="恶名")

            # 师父临死诅咒
            self.db.诅咒 = "师父诅咒: 天劫威力+5%"

        elif 方式 == "叛逃":
            self.db.业力 += 30

        # 清除师徒关系
        self.db.师父 = None

        # 师门追杀令
        self.发布追杀令(发布者=师父, 赏金=10000)

        # 广播
        from evennia import SESSIONS
        SESSIONS.announce_all(f"|r【叛师】{self.key} 背叛了师父 {师父.key}!|n")

        return True

    def 背叛门派(self):
        """背叛门派"""
        if not hasattr(self.db, "门派") or not self.db.门派:
            self.msg("你没有门派")
            return False

        门派 = self.db.门派

        # 记录背叛
        self.db.背叛记录["叛门"] = True

        # 标记
        self.tags.add("叛徒", category="恶名")

        # 后果
        self.db.忠诚度["门派"] = 0
        self.db.声望 -= 300
        self.db.业力 += 50

        # 清除门派
        self.db.门派 = None
        self.tags.remove(门派, category="门派")

        # 门派追杀令
        self.发布追杀令(发布者=门派, 赏金=10000)

        # 广播
        from evennia import SESSIONS
        SESSIONS.announce_all(f"|r【叛门】{self.key} 背叛了 {门派}!|n")

    def 背叛国家(self, 方式="投敌"):
        """背叛国家"""
        if not hasattr(self.db, "国籍") or not self.db.国籍:
            self.msg("你没有国籍")
            return False

        国家 = self.db.国籍

        # 记录背叛
        self.db.背叛记录["叛国"] = True

        # 标记
        self.tags.add("叛国者", category="恶名")

        # 后果
        self.db.忠诚度["国家"] = 0
        self.db.声望 -= 800
        self.db.业力 += 100

        if 方式 == "投敌":
            self.tags.add("卖国贼", category="恶名")
        elif 方式 == "造反":
            self.tags.add("乱臣贼子", category="恶名")

        # 清除国籍
        self.db.国籍 = None

        # 国家通缉
        self.发布追杀令(发布者=国家, 赏金=100000)

        # 连坐家族
        if hasattr(self.db, "家族"):
            # 家族被牵连(需要实现家族系统)

        # 广播
        from evennia import SESSIONS
        SESSIONS.announce_all(f"|R【叛国】{self.key} 背叛了 {国家}!|n")

    def 背叛朋友(self, 朋友):
        """背叛朋友"""
        # 记录
        self.db.背叛记录["叛友次数"] += 0.1
        self.db.义气值 -= 2

        # 如果是结拜兄弟
        if hasattr(self.db, "结拜兄弟") and 朋友 in self.db.结拜兄弟:
            self.db.义气值 -= 5  # 额外扣除
            self.db.结拜兄弟.remove(朋友)
            self.tags.add("背信弃义", category="恶名")
            self.db.业力 += 20

        # 声望下降
        self.db.声望 -= 100

        # 朋友仇恨
        if hasattr(朋友.db, "仇人"):
            朋友.db.仇人.append(self)

        self.msg(f"|r你背叛了 {朋友.key}!|n")
        朋友.msg(f"|R{self.key} 背叛了你!|n")

    def 堕魔(self):

        if self.db.背叛记录["堕魔"]:
            self.msg("你已经是魔修了")
            return False

        # 记录
        self.db.背叛记录["堕魔"] = True

        # 标记
        self.tags.add("魔修", category="阵营")
        self.tags.remove("正道", category="阵营")

        # 外观变化
        self.db.描述 += "\n|r他的眼睛泛着血红色的光芒,周身环绕着黑色魔气。|n"

        # 属性变化
        if hasattr(self.db, "攻击力"):
            self.db.攻击力 = int(self.db.攻击力 * 0.15)
        if hasattr(self.db, "修炼速度"):
            self.db.修炼速度 = int(self.db.修炼速度 * 0.2)

        # 声望变化
        self.db.声望 -= 1000  # 正道声望暴跌

        # 如果有种族声望系统
        if hasattr(self.db, "种族声望"):
            self.db.种族声望["人族"] = -500
            self.db.种族声望["魔族"] = 300

        # 心魔增加
        self.db.心魔 = self.db.get("心魔", 0) + 3

        # 广播
        from evennia import SESSIONS
        SESSIONS.announce_all(f"|R【堕魔】{self.key} 堕入了魔道!|n")

            self.db.追杀令 = []

        追杀令 = {
            "发布者": 发布者,
            "目标": self,
            "赏金": 赏金,
            "状态": "有效",
        }

        self.db.追杀令.append(追杀令)

        self.msg(f"|R【追杀令】{发布者} 发布了对你的追杀令,赏金 {赏金}!|n")

    def 检查背叛标签(self):
        """检查玩家的背叛标签"""
        恶名列表 = []
        for tag in self.tags.get(category="恶名"):
            恶名列表.append(tag)

        if 恶名列表:
            return "、".join(恶名列表)
        else:
            return "无"

    def 获取业力等级(self):
        """获取业力等级"""
        业力 = self.db.业力

        if 业力 <= -100:
            return "大善人(功德无量)"
        elif 业力 <= -50:
            return "善人"
        elif 业力 <= -10:
            return "小善"
        elif -10 < 业力 < 10:
            return "中性"
        elif 业力 < 50:
            return "小恶"
        elif 业力 < 100:
            return "恶人"
        elif 业力 < 300:
            return "大恶人"
        else:
            return "十恶不赦"
```

## 十、平衡性设计

### 1.01 避免滥用

```python
【冷却时间】
- 背叛后无法立即再背叛
- 例如: 叛门后0.1年内无法加入新门派

【递增惩罚】
- 第一次叛友: 义气 -2
- 第二次叛友: 义气 -4
- 第三次叛友: 义气 -8,无人信任

【不可逆后果】
- 某些背叛永久不可逆
- 例如: 杀师后无法救赎
```

### 1.02 背叛的价值

```python
背叛不应该是:
❌ 简单的道德选择题
❌ 纯粹的恶行
❌ 没有意义的数值变化

背叛应该是:
✅ 复杂的道德困境
✅ 有深刻动机的选择
✅ 有意义的剧情分支
✅ 影响玩家体验的重大决定
```
