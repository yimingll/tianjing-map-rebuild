# 修仙游戏货币系统设计方案 v2.0 - 世界观对齐版

> **"灵石为修行之本，功德乃天道之眼。气运定机缘深浅，贡献显门派忠心。"**
>
> 本文档基于《玄鉴仙录·卷五·市井烟火》,定义青玄界真实的货币交易规则

---

## 一、货币体系层级（三界分离）

### 【凡间货币】 - 练气期至筑基期

#### 1. 金银铜（凡人货币）💰

**等级划分**:
- **铜钱** - 最基础货币
- **银两** - 中等货币
- **金锭** - 高级凡间货币

**兑换比例**:
```
100文铜钱 = 1两白银
10两白银 = 1两黄金
```

**用途**:
- 凡人城镇交易
- 购买基础物资（食物、布匹、凡药）
- 雇佣凡人劳工
- 低级灵药采购
- 练气初期弟子日常开销
- 租赁凡人客栈（5-50银两/天）
- 购买凡间秘籍（武功、医术）

**获取方式**:
- 完成凡人任务（+10-100银两）
- 击杀山贼/野兽（+5-50银两）
- 售卖凡品物资
- 门派月俸（外门弟子10-100银两/月）
- 帮助凡人耕作/治病
- 护送商队（+100-1000银两）

**价值参考**:
- 1铜钱 = 1个馒头
- 1银两 = 普通凡人一月生活费
- 1金锭 = 凡人一年收入

**真实案例**:
```
清河城小贩王二,一月收入约二两银子,养活一家五口。
玄天宗外门弟子每月月俸十两银子,足够在城中买房。
```

### 【人界货币】 - 筑基期至化神期

#### 2. 灵石（主流通货币）⭐

这是修仙世界最核心的货币系统！

##### 灵石等级划分

| 等级 | 名称 | 灵气含量 | 兑换比例 | 适用境界 |
|------|------|---------|---------|---------|
| 下品 | 下品灵石 | 基准 | - | 练气期-筑基期 |
| 中品 | 中品灵石 | 100倍 | 1中品 = 100下品 | 筑基期-金丹期 |
| 上品 | 上品灵石 | 10000倍 | 1上品 = 100中品 | 金丹期-元婴期 |
| 极品 | 极品灵石 | 百万倍 | 1极品 = 100上品 | 元婴期-化神期 |

> **重要**: 青玄界灵石兑换比例为100:1,而非其他小世界的10:1!
> 这是因为青玄界灵气浓度极高,导致中品以上灵石价值远超下品。

**兑换体系** (标准100:1比例):
```
1 极品灵石 = 100 上品灵石
1 上品灵石 = 100 中品灵石
1 中品灵石 = 100 下品灵石

完整换算：1极品灵石 = 1,000,000 下品灵石
```

**灵石特性**:
- **可修炼**: 吸收灵石恢复真元/增加修为
- **可交易**: 修士间主要货币
- **有耐久**: 灵气耗尽变为废石
- **有品质**: 同等级灵石有纯度差异（60%-100%纯度）
- **分属性**: 五行灵石（金木水火土），属性对应修士修炼速度更快

**用途分类**:

**交易用途**:
- 修仙坊市购物（功法、法宝、丹药）
- 租赁洞府（10-1000灵石/月）
- 拍卖行竞拍
- 雇佣护法、保镖（100-1000灵石/次）
- 门派贡献度兑换
- 传送阵费用（1-100灵石/次）
- 购买炼丹/炼器材料

**修炼用途**:
- 吸收灵气恢复真元
- 布置聚灵阵（10-1000灵石/阵）
- 炼丹/炼器燃料
- 突破境界辅助（消耗10-1000灵石/次）
- 饲养灵兽（5-50灵石/天）
- 洞府闭关（50-500灵石/天）

**战斗用途**:
- 激活法宝（消耗灵石）
- 阵法能量核心
- 符箓绘制材料
- 战斗中快速恢复真元（1中品灵石 = 30%真元）

**获取方式**:
- 灵石矿脉开采（主要产出）
- 击杀妖兽（掉落10-1000下品）
- 完成任务奖励（100-5000下品）
- 出售物品/装备
- 门派月俸（内门弟子500-5000下品/月）
- 炼丹/炼器售卖产品
- 下副本/秘境掉落
- 种植灵药售卖
- 探索上古洞府（大量灵石宝藏）

**市场价格参考**（基于《玄鉴仙录》真实物价）:

| 物品 | 价格 | 购买地点 |
|------|------|---------|
| 聚气丹（练气期） | 5-20下品 | 白云坊市 |
| 筑基丹（筑基期） | 1000上品 | 灵药谷、拍卖会 |
| 下品法器 | 20-80下品 | 白云坊市 |
| 中品法器 | 80-300下品 | 白云坊市 |
| 上品法剑 | 500上品 | 铸剑山庄 |
| 玄阶下品功法 | 300-1000下品 | 坊市、门派 |
| 洞府租金/月 | 50-500下品 | 各大城市 |
| 灵兽蛋 | 200-2000下品 | 妖兽山脉、坊市 |
| 云来客栈房间/晚 | 1下品 | 清河城 |
| 一壶灵泉酒 | 10下品 | 醉仙居 |

**真实案例**:
```
【筑基丹价格】
葛青云谷主炼制的筑基丹,白云坊市售价1200上品灵石一颗。
外面散修炼制的劣质筑基丹,也要800上品灵石。
这个价格,一般散修要攒十年才买得起。

【灵石矿脉】
玄天宗占据昆仑山脉三条灵脉,每年产出约十万中品灵石。
血河宗为了争夺灵脉,曾与玄天宗大战三百回合。
```

#### 3. 天材地宝（以物易物）🌿

**定义**: 珍稀材料本身可作为货币

**高价值材料示例**:
- **千年灵芝**: 价值1000上品灵石
- **真龙精血**: 价值10极品灵石（传说中的宝物）
- **凤凰羽毛**: 价值5极品灵石
- **九天神雷木**: 价值2极品灵石
- **太古星辰铁**: 价值3极品灵石

**特点**:
- 稀有度极高
- 往往只能以物易物
- 拍卖会硬通货
- 某些交易拒收灵石，只要天材地宝

**用途**:
- 炼制顶级法宝
- 炼制仙丹
- 突破境界瓶颈
- 改造体质
- 延寿（某些可+10-100年寿命）

**真实案例**:
```
【拍卖会传奇】
五百年前,白云坊市拍卖会上,曾出现一滴真龙精血。
最终被血河宗宗主血无涯以十件法宝、五万上品灵石竞得。
据说血无涯用此精血炼制了血河大阵,威力倍增。
```

### 【仙界货币】 - 飞升后（暂不实装）

#### 4. 仙石（仙界流通货币）✨

**定义**: 飞升后在仙界使用的货币

**等级**:
- **下品仙石** = 100极品灵石（灵气浓度）
- **中品仙石** = 100下品仙石
- **上品仙石** = 100中品仙石

**特点**:
- 含仙灵之气（远高于灵气）
- 下界无法使用（会爆裂）
- 仙界独有产物
- 可助真仙修炼
- 普通修士持有会被仙气反噬

**用途**:
- 仙界城池交易
- 购买仙器、仙丹
- 学习仙术
- 租赁仙府
- 修炼仙道功法

**获取方式**:
- 飞升时携带的灵石转化（1000:1损耗）
- 仙界采矿
- 完成仙界任务
- 击杀仙界妖兽

> **注**: 仙石系统暂不实装,待玩家飞升后开启。

## 二、专用货币（特殊兑换货币）

### 5. 贡献度/宗门贡献点 🏛️

**定义**: 门派内部流通的特殊货币，代表对门派的贡献

**获取方式**:
- 完成门派任务（+5-50）
- 上交灵药/材料（价值1:1转化）
- 炼制丹药上交（+10-100）
- 击杀敌对门派修士（+20-200）
- 参与门派战争（+50-500）
- 捐献灵石（**100下品灵石 = 80贡献度**,有20%损耗）
- 教授低级弟子（+1-10/次）
- 为门派炼器/炼丹（+5-50/次）
- 发现新灵脉/秘境（+100-1000）

**用途**:
- 兑换门派功法（100-5000贡献）
  - 黄阶功法: 100-300
  - 玄阶功法: 500-1500
  - 地阶功法: 3000-9999
- 兑换门派丹药（10-500贡献）
- 兑换门派法宝（50-2000贡献）
- 职位晋升（外门→内门需500贡献）
- 租用门派设施:
  - 炼丹房: 10贡献/天
  - 炼器坊: 15贡献/天
  - 藏经阁: 5贡献/天
  - 密室修炼: 20贡献/天（修炼速度×2）
- 购买门派秘籍
- 申请门派资源支持（闭关、渡劫）
- 获得长老指点（50贡献/次）

**特性**:
- **不可交易**: 只能个人使用，无法转让
- **不可继承**: 死亡后清零
- **门派专属**: 更换门派后清零（叛门）

**贡献度等级称号**:
- 0-100: 普通弟子
- 100-500: 优秀弟子
- 500-1500: 精英弟子
- 1500-5000: 核心弟子
- 5000+: 门派栋梁（有资格竞选长老）

**真实案例**:
```
【玄天宗贡献度】
剑峰弟子李云剑,十年内累积3000贡献度。
他用1500贡献度兑换了地阶剑法《斩天剑诀》。
又用1000贡献度申请了掌门云无极的亲自指点。
剩余500贡献度,留着以备不时之需。

【灵药谷贡献度】
外门弟子小花,每天采药上交,一年累积800贡献度。
她用500贡献度晋升内门,又用300贡献度学习了《青木长生诀》。
```

### 6. 功德点（善恶业货币）⚖️

**定义**: 记录善恶行为的特殊数值，影响天劫和气运

**功德值范围**: -9999 ~ +9999

**获取方式（正功德）**:

**善业来源**:
- 救助凡人（+1-10/人）
- 击杀邪修（+5-50/人）
- 除魔卫道（+10-100/次）
- 解救被困修士（+20）
- 捐赠给穷人（+1功德/1000下品灵石）
- 传授凡人武艺（+5/人）
- 制止杀戮（+10）
- 保护凡人村庄不受妖兽侵扰（+50）
- 炼制救命丹药赠予他人（+5-20）
- 化解修士间仇怨（+10-50）

**恶业来源（负功德）**:
- 滥杀无辜（-10/人）
- 屠城灭族（-1000）
- 炼制邪器（-50）
- 夺舍重生（-100）
- 血祭术（-20/次）
- 炼制邪丹（需活人炼制）（-100）
- 背叛师门（-200）
- 杀害同门（-50-200）
- 残害灵兽（-5-50）

**功德效果**:

**正功德效果**:
- 功德>100: 天劫难度-10%
- 功德>500: 顿悟几率+5%，NPC好感度+20%
- 功德>1000: 突破成功率+10%，可感悟天道
- 功德>5000: 天道庇护，遭遇致命危险有1%几率自动化解
- 功德>9999: 功德成圣，可兑换天道奖励（功法、法宝、气运）

**负功德效果**:
- 功德<-100: 天劫难度+20%，易走火入魔
- 功德<-500: 正道追杀令，进入正道城镇会被攻击
- 功德<-1000: 天谴降临，随机灾祸
- 功德<-5000: 心魔缠身，修炼速度-50%
- 功德<-9999: 天道抹杀，引发灭杀天劫

**功德消耗**:
- 消除因果（100功德 = 1因果）
- 请求天道帮助渡劫（-500功德，天劫难度-30%）
- 复活重生（-1000功德）
- 改变命格（-2000功德）
- 赎罪转化恶业（2:1，200正功德消除100恶业）

**特殊**:
- 死亡后功德不清零
- 影响转世投胎（正功德高可投胎到修仙世家）
- 可通过立地成佛、放下屠刀获得大量功德

**真实案例**:
```
【血无涯的恶业】
血河宗宗主血无涯,修炼血道功法,杀戮无数。
其功德值据说已跌破-5000,每次渡劫都九死一生。
但血无涯不以为意,反而说:"功德?那是弱者的枷锁!"

【大雷音寺觉明大师】
大雷音寺主持觉明大师,一生救人无数,功德值超过8000。
他渡化神劫时,天劫威力只有正常的一半。
据说觉明大师即将功德圆满,有望功德成圣。
```

### 7. 气运值（隐藏货币）🎲

**定义**: 影响机缘和运气的隐藏数值，修仙之路的"幸运值"

**初始值**: 创建角色随机（5-15）

**气运值范围**: 0-100

**获取方式**:
- 完成特殊机缘（+1-5）
- 佩戴气运法宝（+临时气运，如玉佩、福袋）
- 服用气运丹（+1-3，极其稀有）
- 击杀"气运之子"（+气运掠夺，5-10点）
- 获得天命眷顾（+10，极罕见）
- 完成隐藏成就（+1-3）
- 救助有大气运的人（+2-5）
- 获得上古传承（+3-10）
- 正功德>5000（+5气运）

**消耗方式**:
- 触发传说级机缘（-10）
- 强行改命（-20）
- 使用逆天宝物（-5）
- 遭遇劫难（-1-5）
- 被他人掠夺（被杀）
- 天劫失败（-5）

**气运等级划分**:
- 0-10: 霉运缠身（厄运之人）
- 11-30: 运气欠佳
- 31-50: 普通水平
- 51-70: 运气不错
- 71-90: 鸿运当头
- 91-100: 气运之子（天命之人）

**气运影响**:
- 暴击率: 气运相关
- 掉落率: 气运相关
- 机缘触发概率: 气运越高越容易触发
- 突破成功率: 气运影响突破
- 天劫难度: 气运越高，天劫越强（天妒英才）
  - 气运<30: 天劫难度-20%
  - 气运>70: 天劫难度+30%
- 炼丹/炼器暴击: 气运影响出极品概率
- 获得稀有宝物概率: 气运影响

**特殊机制**:
- **气运掠夺**: 击杀其他修士可掠夺其10%-30%气运
- **气运转移**: 某些秘术可将气运转给他人
- **气运共享**: 道侣、师徒可共享部分气运（+5%）

**气运显示**:
- 玩家可见自己气运值
- 修炼"望气术"可查看他人气运（神识检定）
- 气运极高者（>90）有金光环绕

**真实案例**:
```
【独孤求败的气运】
散修联盟盟主独孤求败,气运据说高达85。
他年少时曾遇剑魔前辈,得玄铁重剑、剑道传承。
后又在绝情谷底发现神雕,习得独孤九剑。
江湖人称:"独孤之幸,天命所归。"

【倒霉的散修王大锤】
散修王大锤,气运只有5,一生坎坷。
他曾在秘境中找到宝藏,结果被路过的金丹修士抢走。
又曾炼丹炸炉三次,损失惨重。
最后王大锤感叹:"这辈子,就是来还债的!"
```

### 9. 悟道点（修炼货币）📿

**定义**: 感悟天地大道获得的特殊点数

**获取方式**:
- 顿悟时获得（+1-10）

## 五、特色系统

### 1. 钱庄/灵石票庄系统 🏦

**青玄界三大商帮**:
- **晋商** - 根据地在中州,经营灵石票庄
- **徽商** - 根据地在江南,经营丹药买卖
- **粤商** - 根据地在南疆,经营妖兽材料

**晋商钱庄功能**:

#### A. 存储灵石
- 安全保管（防盗、防抢）
- 收取保管费（1%/年）
- 跨城取款（手续费2%）
- 利息收益（0.5%/年，定期存储）

#### B. 开具票据
- 大额灵石可开具票据
- 方便远程交易
- 防止被劫
- 票据可转让

#### C. 贷款系统
- 可向钱庄贷款
- 利率: 20%/年
- 需要抵押（法宝、洞府、功法）
- 逾期会被追债（NPC追杀）
- 最高贷款额度: 信用等级×1000灵石

**信用等级**:
- 根据还款记录、声望、境界综合评定
- 1-10级，等级越高额度越高
- 违约会降低信用等级

**真实案例**:
```
【晋商首富】

连玄天宗都曾向他借过灵石。

散修李赌徒为了买筑基丹,向钱庄贷款10000下品灵石。
结果筑基失败,还不起钱,被晋商派来的金丹护卫追杀。
最终李赌徒只能卖身为奴,给钱庄打工还债。
```

### 2. 货币通缩/通胀机制

**通缩措施**（货币回收）:
- 装备强化失败（灵石消失）
- 炼丹失败（材料+灵石消失）
- NPC商店高价商品（系统回收）
- 天劫渡劫消耗（丹药、阵法）
- 建设门派/势力设施（大额消耗）
- 购买洞府产权（永久消耗）
- 死亡掉落（部分灵石消失）
- 传送阵费用
- 喂养灵兽（持续消耗）

**产出控制**:
- 灵石矿脉产出有CD
- 高级货币获取困难
- BOSS刷新时间长
- 副本每日次数限制

**动态调节**:
- 系统监测货币总量
- 产出过多时降低掉落率
- 货币稀缺时增加任务奖励

- 灵石矿产出CD

- 交易冷却时间

- 佣兵任务收益
- 发现秘境奖励

## 九、货币系统与其他系统联动

### 与境界系统
- 境界越高，货币产出越多
- 境界突破需要大量灵石（消耗点）

### 与装备系统
- 装备强化消耗灵石
- 装备分解获得材料
- 购买装备需要灵石

### 与门派系统
- 贡献度兑换门派资源
- 门派月俸（固定收入）
- 门派任务（贡献度来源）

### 与机缘系统
- 气运影响机缘触发
- 机缘获得大量货币
- 功德影响机缘质量

### 与时间系统
- 货币利息随时间增长
- 贷款利息随时间累积
- 月俸按月发放

## 十、经典货币事件

### 1. 拍卖会系统
**触发**: 每月1号，各大城池举办

**拍卖物品**:
- 极品法宝（起拍价: 1000上品灵石）
- 地阶功法（起拍价: 5000上品灵石）
- 天材地宝（起拍价: 100上品灵石）
- 灵兽蛋（起拍价: 500上品灵石）
- 特殊洞府产权（起拍价: 9999上品灵石）

**竞拍规则**:
- 每次加价不少于起拍价10%
- 最高价者得
- 需预先缴纳保证金（起拍价50%）
- 流拍退还保证金

**真实案例**:
```
【白云坊市拍卖会】
五百年前,白云坊市拍卖会上,曾拍卖一把仙剑残片。
起拍价五万上品灵石,最终被血无涯以十万上品拍得。
据说血无涯用此仙剑残片,炼制了血河宗镇宗之宝"血煞剑"。

【散修的逆袭】
散修独孤求败,曾在拍卖会上捡漏。
他用1000上品灵石,拍得一把"废剑"。
结果那是上古神剑"玄铁重剑",只是蒙尘太久无人识货。
独孤求败凭此剑,一路逆袭,最终成为散修联盟盟主。
```

**背景**: 发现新灵石矿脉

```
此战后,玄天宗占据昆仑山脉三条灵脉,成为青玄界第一大宗。
```
