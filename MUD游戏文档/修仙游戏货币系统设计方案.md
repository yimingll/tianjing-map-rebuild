# 修仙游戏货币系统设计方案

> 基于《凡人修仙传》《一念逍遥》等经典修仙作品的货币体系设定
>
> **注**: 本方案为纯单机/买断制游戏设计，无充值系统

---

## 一、货币体系层级（三界分离）

### 【凡间货币】 - 练气期至筑基期

#### 1. 金银铜（凡人货币）💰

**等级划分**:
- **铜钱** - 最基础货币
- **银两** - 中等货币
- **金锭** - 高级凡间货币

**兑换比例**:
```
10铜钱 = 1银两
10银两 = 1金锭
```

**用途**:
- 凡人城镇交易
- 购买基础物资（食物、布匹、凡药）
- 雇佣凡人劳工
- 低级灵药采购
- 练气初期弟子日常开销
- 租赁凡人客栈（5-50银两/天）
- 购买凡间秘籍（武功、医术）

**获取方式**:
- 完成凡人任务（+10-100银两）
- 击杀山贼/野兽（+5-50银两）
- 售卖凡品物资
- 门派月俸（外门弟子10-100银两/月）
- 帮助凡人耕作/治病
- 护送商队（+100-1000银两）

**价值参考**:
- 1铜钱 = 1个馒头
- 1银两 = 普通凡人一月生活费
- 1金锭 = 凡人一年收入

### 【人界货币】 - 筑基期至化神期

#### 2. 灵石（主流通货币）⭐

这是修仙世界最核心的货币系统！

##### 灵石等级划分

| 等级 | 名称 | 灵气含量 | 兑换比例 | 适用境界 |
|------|------|---------|---------|---------|
| 下品 | 下品灵石 | 10单位 | - | 练气期-筑基期 |
| 中品 | 中品灵石 | 100单位 | 10下品 = 1中品 | 筑基期-金丹期 |
| 上品 | 上品灵石 | 1000单位 | 10中品 = 1上品 | 金丹期-元婴期 |
| 极品 | 极品灵石 | 10000单位 | 10上品 = 1极品 | 元婴期-化神期 |
| 灵晶 | 灵晶 | 100000单位 | 10极品 = 1灵晶 | 化神期+ |

**兑换体系** (标准10:1比例):
```
1 灵晶 = 10 极品灵石
1 极品灵石 = 10 上品灵石
1 上品灵石 = 10 中品灵石
1 中品灵石 = 10 下品灵石

完整换算：1灵晶 = 10,000 下品灵石
```

**灵石特性**:
- **可修炼**: 吸收灵石恢复真元/增加修为
- **可交易**: 修士间主要货币
- **有耐久**: 灵气耗尽变为废石
- **有品质**: 同等级灵石有纯度差异（60%-100%纯度）
- **分属性**: 五行灵石（金木水火土），属性对应修士修炼速度+20%

**用途分类**:

**交易用途**:
- 修仙坊市购物（功法、法宝、丹药）
- 租赁洞府（10-1000灵石/月）
- 拍卖行竞拍
- 雇佣护法、保镖（100-1000灵石/次）
- 门派贡献度兑换
- 传送阵费用（1-100灵石/次）
- 购买炼丹/炼器材料

**修炼用途**:
- 吸收灵气恢复真元
- 布置聚灵阵（10-1000灵石/阵）
- 炼丹/炼器燃料
- 突破境界辅助（消耗10-1000灵石/次）
- 饲养灵兽（5-50灵石/天）
- 洞府闭关（50-500灵石/天）

**战斗用途**:
- 激活法宝（消耗灵石）
- 阵法能量核心
- 符箓绘制材料
- 战斗中快速恢复真元（1中品灵石 = 30%真元）

**获取方式**:
- 灵石矿脉开采（主要产出）
- 击杀妖兽（掉落10-1000下品）
- 完成任务奖励（100-5000下品）
- 出售物品/装备
- 门派月俸（内门弟子500-5000下品/月）
- 炼丹/炼器售卖产品
- 下副本/秘境掉落
- 种植灵药售卖
- 探索上古洞府（大量灵石宝藏）

**市场价格参考**（下品灵石）:
- 聚气丹（练气期）: 5-20
- 筑基丹（筑基期）: 50-200
- 下品法器: 20-80
- 中品法器: 80-300
- 玄阶下品功法: 300-1000
- 洞府租金/月: 50-500
- 灵兽蛋: 200-2000

#### 3. 天材地宝（以物易物）🌿

**定义**: 珍稀材料本身可作为货币

**高价值材料示例**:
- **千年灵芝**: 价值1000上品灵石
- **真龙精血**: 价值10极品灵石
- **凤凰羽毛**: 价值5极品灵石
- **九天神雷木**: 价值2极品灵石
- **太古星辰铁**: 价值3极品灵石

**特点**:
- 稀有度极高
- 往往只能以物易物
- 拍卖会硬通货
- 某些交易拒收灵石，只要天材地宝

**用途**:
- 炼制顶级法宝
- 炼制仙丹
- 突破境界瓶颈
- 改造体质
- 延寿（某些可+10-100年寿命）

### 【仙界货币】 - 渡劫期至真仙

#### 4. 仙石（仙界流通货币）✨

**定义**: 飞升后在仙界使用的货币

**等级**:
- **下品仙石** = 100极品灵石（灵气浓度）
- **中品仙石** = 10下品仙石
- **上品仙石** = 10中品仙石
- **仙髓** = 10上品仙石（极品仙石）

**特点**:
- 含仙灵之气（远高于灵气）
- 下界无法使用（会爆裂）
- 仙界独有产物
- 可助真仙修炼
- 普通修士持有会被仙气反噬

**用途**:
- 仙界城池交易
- 购买仙器、仙丹
- 学习仙术
- 租赁仙府
- 修炼仙道功法

**获取方式**:
- 飞升时携带的灵石转化（1000:1损耗）
- 仙界采矿
- 完成仙界任务
- 击杀仙界妖兽

## 二、专用货币（特殊兑换货币）

### 5. 贡献度/宗门贡献点 🏛️

**定义**: 门派内部流通的特殊货币，代表对门派的贡献

**获取方式**:
- 完成门派任务（+5-50）
- 上交灵药/材料（价值1:1转化）
- 炼制丹药上交（+10-100）
- 击杀敌对门派修士（+20-200）
- 参与门派战争（+50-500）
- 捐献灵石（**1下品灵石 = 1贡献度**，门派收购价与市场价持平）
- 教授低级弟子（+1-10/次）
- 为门派炼器/炼丹（+5-50/次）
- 发现新灵脉/秘境（+100-1000）

> **重要经济平衡**: 详见《【重要】文档统一标准.md》第六章 - 经济平衡指导原则
> - 门派贡献度兑换商品价格应为市场价的80-120%
> - 避免通过门派系统无限套利破坏经济平衡

**用途**:
- 兑换门派功法（100-5000贡献）
  - 黄阶功法: 100-300
  - 玄阶功法: 500-1500
  - 地阶功法: 3000-9999
- 兑换门派丹药（10-500贡献）
- 兑换门派法宝（50-2000贡献）
- 职位晋升（外门→内门需500贡献）
- 租用门派设施:
  - 炼丹房: 10贡献/天
  - 炼器坊: 15贡献/天
  - 藏经阁: 5贡献/天
  - 密室修炼: 20贡献/天（修炼速度×2）
- 购买门派秘籍
- 申请门派资源支持（闭关、渡劫）
- 获得长老指点（50贡献/次）

**特性**:
- **不可交易**: 只能个人使用，无法转让
- **不可继承**: 死亡后清零
- **门派专属**: 更换门派后清零（叛门）

**贡献度等级称号**:
- 0-100: 普通弟子
- 100-500: 优秀弟子
- 500-1500: 精英弟子
- 1500-5000: 核心弟子
- 5000+: 门派栋梁（有资格竞选长老）

### 6. 功德点（善恶业货币）⚖️

**定义**: 记录善恶行为的特殊数值，影响天劫和气运

**功德值范围**: -9999 ~ +9999

**获取方式（正功德）**:

**善业来源**:
- 救助凡人（+1-10/人）
- 击杀邪修（+5-50/人）
- 除魔卫道（+10-100/次）
- 解救被困修士（+20）
- 捐赠给穷人（+1功德/10灵石）
- 传授凡人武艺（+5/人）
- 制止杀戮（+10）
- 保护凡人村庄不受妖兽侵扰（+50）
- 炼制救命丹药赠予他人（+5-20）
- 化解修士间仇怨（+10-50）

**恶业来源（负功德）**:
- 滥杀无辜（-10/人）
- 屠城灭族（-1000）
- 炼制邪器（-50）
- 夺舍重生（-100）
- 血祭术（-20/次）
- 炼制邪丹（需活人炼制）（-100）
- 背叛师门（-200）
- 杀害同门（-50-200）
- 残害灵兽（-5-50）

**功德效果**:

**正功德效果**:
- 功德>100: 天劫难度-10%
- 功德>500: 顿悟几率+5%，NPC好感度+20%
- 功德>1000: 突破成功率+10%，可感悟天道
- 功德>5000: 天道庇护，遭遇致命危险有1%几率自动化解
- 功德>9999: 功德成圣，可兑换天道奖励（功法、法宝、气运）

**负功德效果**:
- 功德<-100: 天劫难度+20%，易走火入魔
- 功德<-500: 正道追杀令，进入正道城镇会被攻击
- 功德<-1000: 天谴降临，随机灾祸
- 功德<-5000: 心魔缠身，修炼速度-50%
- 功德<-9999: 天道抹杀，引发灭杀天劫

**功德消耗**:
- 消除因果（100功德 = 1因果）
- 请求天道帮助渡劫（-500功德，天劫难度-30%）
- 复活重生（-1000功德）
- 改变命格（-2000功德）
- 赎罪转化恶业（2:1，200正功德消除100恶业）

**特殊**:
- 死亡后功德不清零
- 影响转世投胎（正功德高可投胎到修仙世家）
- 可通过立地成佛、放下屠刀获得大量功德

### 7. 气运值（隐藏货币）🎲

**定义**: 影响机缘和运气的隐藏数值，修仙之路的"幸运值"

**初始值**: 创建角色随机（5-15）

**气运值范围**: 0-100

**获取方式**:
- 完成特殊机缘（+1-5）
- 佩戴气运法宝（+临时气运，如玉佩、福袋）
- 服用气运丹（+1-3，极其稀有）
- 击杀"气运之子"（+气运掠夺，5-10点）
- 获得天命眷顾（+10，极罕见）
- 完成隐藏成就（+1-3）
- 救助有大气运的人（+2-5）
- 获得上古传承（+3-10）
- 正功德>5000（+5气运）

**消耗方式**:
- 触发传说级机缘（-10）
- 强行改命（-20）
- 使用逆天宝物（-5）
- 遭遇劫难（-1-5）
- 被他人掠夺（被杀）
- 天劫失败（-5）

**气运等级划分**:
- 0-10: 霉运缠身（厄运之人）
- 11-30: 运气欠佳
- 31-50: 普通水平
- 51-70: 运气不错
- 71-90: 鸿运当头
- 91-100: 气运之子（天命之人）

**气运影响**:
- 暴击率: 气运×0.5%
- 掉落率: 气运×2%
- 机缘触发概率: 气运×3%
- 突破成功率: 气运×1%
- 天劫难度: 气运越高，天劫越强（天妒英才）
  - 气运<30: 天劫难度-20%
  - 气运>70: 天劫难度+30%
- 炼丹/炼器暴击: 气运×1%（出极品概率）
- 获得稀有宝物概率: 气运×5%

**特殊机制**:
- **气运掠夺**: 击杀其他修士可掠夺其10%-30%气运
- **气运转移**: 某些秘术可将气运转给他人
- **气运共享**: 道侣、师徒可共享部分气运（+5%）

**气运显示**:
- 玩家可见自己气运值
- 修炼"望气术"可查看他人气运（神识检定）
- 气运极高者（>90）有金光环绕

### 8. 战功（国家战争货币）⚔️

**定义**: 参与国家战争获得的特殊货币

**获取方式**:
- 击杀敌国修士（+1-10/人，根据境界）
- 占领城池（+50-500）
- 破坏敌方设施（+10-50）
- 保卫己方城池（+20）
- 战争胜利（全员+100）
- 击杀敌方将领（+100-500）
- 斩首敌方大将（+1000）
- 参与攻城战（+50）
- 夺取敌方资源点（+30-200）

**用途**:
- 兑换军功法宝（100-1000战功）
  - 军功剑（专属武器）: 500战功
  - 护国战甲: 800战功
  - 破军旗（战阵法宝）: 1500战功
- 兑换战争丹药（10-100战功）
  - 军粮丸（快速恢复）: 10战功
  - 狂暴丹（临时增强）: 50战功
- 兑换军衔提升（50战功/级）
  - 士兵 → 百夫长 → 千夫长 → 将军 → 大将军
- 兑换特殊称号（500战功）
  - "百人斩"、"破城勇士"、"国之柱石"
- 兑换国家资源配额
- 获得封地（1000战功可获得城池管理权）
- 国家月俸增加（战功越高，俸禄越高）

**军衔系统**:
| 军衔 | 所需战功 | 月俸 | 特权 |
|------|---------|------|------|
| 士兵 | 0 | 10灵石 | 无 |
| 百夫长 | 50 | 30灵石 | 可指挥100人 |
| 千夫长 | 300 | 100灵石 | 可指挥1000人 |
| 将军 | 1000 | 300灵石 | 可申请国家资源 |
| 大将军 | 5000 | 1000灵石 | 参与国家决策 |

**战功清零**:
- 每次战争结束后可选择：
  1. 保留战功（下次战争继续累积）
  2. 清零兑换奖励（获得额外20%兑换加成）

### 11. 魂石/元神碎片（PVP货币）👻

**定义**: 击杀敌人或分解装备获得

**获取方式**:
- 击杀玩家/强敌（PK）获得（+1-10）
- 分解紫色以上装备（+5-50）
- 击杀BOSS（+10-100）
- 分解法宝（+20-200）
- 渡劫成功获得（+50）
- 探索古战场（+5-50）

**用途**:
- 兑换PVP专属装备
  - 杀戮战甲（PVP伤害+20%）: 500魂石
  - 血饮剑（击杀回血）: 800魂石
- 兑换元神技能
  - 元神分身: 1000魂石
  - 元神离体: 1500魂石
- 强化元神（+神识强度）
  - 每100魂石提升1%神识
- 购买复活道具
  - 复活符（原地满血复活）: 500魂石
  - 替身傀儡（代替死亡）: 1000魂石

## 四、货币兑换系统

```
1000金锭 = 1下品灵石（单向，灵石不能换回金银）

10下品灵石 = 1中品灵石

10极品灵石 = 1灵晶

**灵石 → 贡献度**
```

仅能单向兑换，不可逆
```

**功德 → 气运**
```

## 五、特色系统

### 1. 钱庄/灵石票庄系统 🏦

**功能**:

#### A. 存储灵石
- 安全保管（防盗、防抢）
- 收取保管费（1%/年）
- 跨城取款（手续费2%）
- 利息收益（0.5%/年，定期存储）

#### B. 开具票据
- 大额灵石可开具票据
- 方便远程交易
- 防止被劫
- 票据可转让

#### C. 贷款系统
- 可向钱庄贷款
- 利率: 20%/年
- 需要抵押（法宝、洞府、功法）
- 逾期会被追债（NPC追杀）
- 最高贷款额度: 信用等级×1000灵石

**信用等级**:
- 根据还款记录、声望、境界综合评定
- 1-10级，等级越高额度越高
- 违约会降低信用等级

**示例**:
```
玩家存入1000中品灵石到钱庄
- 活期: 无利息，可随时取出，保管费10中品/年
- 定期1年: 利息5中品，到期取出
- 贷款: 抵押上品法器，可贷500中品灵石
```

---

### 2. 货币通缩/通胀机制

**通缩措施**（货币回收）:
- 装备强化失败（灵石消失）
- 炼丹失败（材料+灵石消失）
- NPC商店高价商品（系统回收）
- 天劫渡劫消耗（丹药、阵法）
- 建设门派/势力设施（大额消耗）
- 购买洞府产权（永久消耗）
- 死亡掉落（部分灵石消失）
- 传送阵费用
- 喂养灵兽（持续消耗）

**产出控制**:
- 灵石矿脉产出有CD
- 高级货币获取困难
- BOSS刷新时间长
- 副本每日次数限制

**动态调节**:
- 系统监测货币总量
- 产出过多时降低掉落率
- 货币稀缺时增加任务奖励

---

### 3. 黑市系统 🌑

**定义**: 非法交易场所，可购买违禁品

**特殊货币**: 暗币（黑市专用）

**获取暗币**:
- 完成灰色任务（+1-10暗币）
- 击杀通缉犯（+5暗币）
- 售卖赃物（+1暗币/100灵石价值）
- 做黑市任务（+10-50暗币）
- 参与走私（+20-100暗币）

**黑市商品**:
- 禁术功法（魔道、鬼修）
- 违禁丹药（狂化丹、夺命丹）
- 偷盗来的宝物（低价出售）
- 雇佣杀手（100-1000暗币）
- 洗白罪恶值（50暗币 = -10恶业）
- 假身份令牌（潜入敌对势力）
- 毒药、暗器

**黑市位置**:
- 隐蔽，需要特定信物进入
- 每周更换地点
- 被正道围剿风险

**风险**:
- 可能遇到正道扫黑
- 商品质量无保证
- 可能被骗
- 被发现会增加恶业

---

### 4. 货币保管与防盗

**储物戒指**:
- 携带上限（根据品质）
  - 下品储物戒: 1000下品灵石
  - 中品储物戒: 9999下品灵石
  - 上品储物戒: 9999中品灵石
- 死亡可能掉落（30%概率）
- 需要认主绑定
- 可设置防盗阵法（需消耗灵石）

**洞府保险箱**:
- 最安全的存储方式
- 死亡不掉落
- 需要自己的洞府
- 可设置多重禁制

**被抢劫风险**:
- 野外被击杀，掉落10%-50%携带灵石
- 城镇安全区不会被抢
- 雇佣护卫可降低风险（10灵石/小时）

---

### 5. 遗产继承系统

**死亡后货币处理**:

| 货币类型 | 继承规则 |
|---------|---------|
| 金银铜 | 100%继承（储物戒/洞府中） |
| 灵石 | 80%继承（野外死亡掉落20%） |
| 贡献度 | 清零，不继承 |
| 功德/恶业 | 100%继承（影响转世） |
| 气运 | 50%继承 |
| 战功 | 清零 |
| 机缘点 | 70%继承 |
| 魂石 | 100%继承 |
| 悟道点 | 50%继承 |
| 声望 | 根据死因，保留30%-70% |

**转世机制**:
- 保留部分货币（见上表）
- 功德影响转世出身
  - 正功德>1000: 转世修仙世家，初始资源丰厚
  - 正功德 0-1000: 正常转世
  - 负功德<-500: 转世贫民/妖族
- 可能觉醒前世记忆（恢复部分资源）

## 八、防沉迷与平衡机制

### 1. 防止通货膨胀
- 高级货币产出稀少

- 死亡惩罚（掉落货币）
- NPC商店价格动态调整

- 新手保护期（前3个月不掉落）
- 师徒系统（师父可资助徒弟）
- 门派救济（贫困弟子补助）
- 任务保底收益

### 3. 防止恶意刷钱
- 每日副本次数限制
- 灵石矿产出CD
- BOSS刷新时间
- 交易冷却时间

### 4. 鼓励多样化赚钱
- 炼丹/炼器收益
- 种植灵药收益
- 佣兵任务收益
- 参与国战收益
- 发现秘境奖励

## 九、货币系统与其他系统联动

### 与境界系统
- 境界越高，货币产出越多
- 境界突破需要大量灵石（消耗点）

### 与装备系统
- 装备强化消耗灵石
- 装备分解获得魂石
- 购买装备需要灵石

### 与门派系统
- 贡献度兑换门派资源
- 门派月俸（固定收入）
- 门派任务（贡献度来源）

### 与国家系统
- 战功兑换军功装备
- 国家月俸（根据军衔）
- 国战参与获得战功

### 与机缘系统
- 气运影响机缘触发
- 机缘获得大量货币
- 功德影响机缘质量

### 与时间系统
- 货币利息随时间增长
- 贷款利息随时间累积
- 月俸按月发放
