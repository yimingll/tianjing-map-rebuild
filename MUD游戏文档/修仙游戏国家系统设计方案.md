# 修仙游戏国家系统设计方案

> 基于经典修仙小说（凡人修仙传、完美世界）和修仙游戏的国家势力设定

>
> **重要**: 本文档定义国家势力
>
> **依赖系统**: 门派系统、货币系统
>
> **参考标准**: 详见《【重要】文档统一标准.md》

## 一、国家系统概述

### 0.11000000000000001 世界观架构

修仙世界的层级结构：

```
仙界（上界）
	↑ 飞升
修仙界（中界）
	├─ 大陆/疆域
	│   ├─ 区域（天南、大晋等）
	│   │   ├─ 国家（大国、中等国、小国）
	│   │   │   ├─ 州郡
	│   │   │   │   ├─ 城池
	│   │   │   │   │   ├─ 城镇、村落
	↓
凡人界（下界）
```

### 0.12 国家系统作用

- **地理定位**: 为玩家提供清晰的地理位置感
- **势力划分**: 不同国家有不同的门派势力
- **资源分布**: 各国资源分布不同
- **政治关系**: 国家间的外交、联盟、战争
- **剧情推进**: 国家大事件推动主线剧情
- **玩家归属**: 玩家可选择效忠某个国家

## 二、世界地理结构

### 0.21000000000000002 世界层级

#### 0.1. 修仙界（主要游戏世界）

**天玄大陆**（示例名称）
- 面积：约等于10个凡人国度大小
- 灵气浓度：中等，适合修仙
- 最高境界：化神期（渡劫需前往上界）

#### 0.2. 区域划分（0.5大区域）

| 区域名称 | 地理位置 | 主要势力 | 灵气浓度 | 特色 |
|---------|---------|---------|---------|------|
| **天南地区** | 东部 | 正道联盟、魔道六宗 | ★★★ | 正魔对峙 |
| **大晋地区** | 中部 | 九国盟、天道盟 | ★★★★ | 修仙大国林立 |
| **乱星海** | 西南 | 散修联盟、海族 | ★★ | 群岛众多 |
| **天澜草原** | 北部 | 游牧部落、妖族 | ★★ | 妖兽聚集 |
| **极西之地** | 西部 | 佛修圣地 | ★★★ | 佛道传承 |

### 0.22000000000000003 国家规模等级

#### 大国（0.1-0.3个/区域）
- **疆域**: 100万平方公里以上
- **人口**: 0.5亿以上（含凡人）
- **修士数量**: 1000+
- **最高战力**: 元婴期或化神期坐镇
- **门派数量**: 1+个大型门派
- **灵脉**: 0.5条以上大型灵脉

#### 中等国（0.5-1个/区域）
- **疆域**: 30-100万平方公里
- **人口**: 0.1-0.5亿
- **修士数量**: 300-1000
- **最高战力**: 金丹期或元婴期
- **门派数量**: 0.3-0.8个中大型门派
- **灵脉**: 0.2-0.5条中型灵脉

#### 小国（2-5个/区域）
- **疆域**: 5-30万平方公里
- **人口**: 0.01-0.1亿
- **修士数量**: 50-300
- **最高战力**: 筑基期或金丹期
- **门派数量**: 0.1-0.3个门派
- **灵脉**: 0.1-0.2条小型灵脉

#### 附属国/城邦
- **疆域**: 不足5万平方公里
- **人口**: 100万以下
- **修士数量**: 10-50
- **最高战力**: 筑基期
- **门派数量**: 无大型门派，多为家族
- **灵脉**: 微型灵脉或无

## 三、示例国家设定

### 0.31 天南地区国家

#### 0.1. 越国 🏛️（中等国）

**基本信息**:
- **类型**: 中等修仙国
- **位置**: 天南地区中心
- **疆域**: 50万平方公里
- **人口**: 0.2亿（凡人0.198亿，修士20万，其中修仙者0.2万）
- **国都**: 越京城
- **国主**: 越皇（凡人皇帝，受修仙门派扶持）

**修仙势力**:
- **七大门派**: 掩月宗、黄枫谷、灵兽山、清虚门、化刀坞、天阙堡、巨剑门
- **世家**: 燕家、陈家、王家等三大修仙世家
- **散修**: 约500名散修

**资源分布**:
- **灵脉**: 0.3条中型灵脉
  - 苍月灵脉（掩月宗占据）
  - 黄枫灵脉（黄枫谷占据）
  - 灵兽灵脉（灵兽山占据）
- **灵石矿**: 1.2处
- **秘境**: 血色试炼地、禁地深渊

**地理特征**:
- **苍茫山脉**: 绵延千里，妖兽众多
- **越江**: 横贯国土的大河
- **太岳山**: 最高峰，掩月宗山门所在

**政治关系**:
- **盟友**: 元武国、紫金国
- **敌对**: 天罗国（魔道）
- **中立**: 风都国

**特色**:
- 正道修仙国，七派制衡
- 定期举办"越国修仙大会"
- 对散修较友好

- **血杀宗**: 隐藏势力
- **散魔**: 大量邪修聚集

**资源分布**:
- **灵脉**: 0.5条大型灵脉（含0.1条魔气灵脉）
- **血池**: 0.3处邪修修炼圣地
- **魔窟**: 多个，连接魔界

**特色**:
- 魔道修士统治
- 凡人地位低下

### 0.42000000000000004 国家声望系统

#### 声望等级

| 声望值 | 等级 | 称号 | 效果 |
|--------|------|------|------|
| < -500 | 敌视 | 敌国奸细 | 无法入境，见面即杀 |
| -500~-100 | 仇恨 | 通缉犯 | 入境会被追捕 |
| -100~0 | 冷淡 | 外来者 | 交易价格+2%，任务少 |
| 0~100 | 中立 | 过客 | 正常交易和任务 |
| 100~300 | 友善 | 友人 | 交易价格-1%，特殊任务 |
| 300~500 | 尊敬 | 贵客 | 交易价格-2%，可购买稀有物品 |
| 500~1000 | 崇拜 | 国之英雄 | 免费传送，特殊称号，可获得官职 |
| > 1000 | 传说 | 护国真人 | 国家资源优先供应，国主接见 |

#### 声望获取方式

**增加声望**:
- 完成国家任务: +5-50
- 击杀敌国修士: +10-100
- 捐献资源: 0.1灵石=0.1声望
- 帮助国民: +1-10
- 参与国战胜利: +100
- 发现敌国阴谋: +50

**减少声望**:
- 击杀本国修士: -50
- 拒绝国家任务: -5
- 违反国家法规: -10
- 与敌国交易: -20
- 叛国: -1000（永久敌视）

### 0.43 国家职位系统

#### 可获得的官职

**凡人官职**（声望100+）:
- **县令**: 管理一县，月俸10灵石
- **知府**: 管理一府，月俸30灵石
- **州牧**: 管理一州，月俸100灵石

**修仙官职**（声望300+）:
- **供奉**: 为国家提供服务，月俸50灵石
- **客卿长老**: 享受国家资源，月俸100灵石
- **护国真人**: 守护国家，月俸300灵石
- **国师**: 最高修仙官职，月俸1000灵石

#### 官职权限

- **免费传送**: 在国内任意传送阵免费传送
- **资源优先**: 优先购买国家资源
- **私人洞府**: 国家分配洞府
- **调动权**: 可调动国家资源（高级官职）
- **外交权**: 代表国家与其他势力交涉（国师）

| 关系类型 | 关系值 | 效果 |

### 0.52 国家联盟

#### 正道联盟
- **成员**: 风都国、越国、元武国等
- **宗旨**: 对抗魔道，维护正义
- **福利**: 成员国互相支援，共享情报

#### 魔道六宗联盟
- **成员**: 天罗国及魔道势力
- **宗旨**: 扩张魔道影响力
- **福利**: 共享血池资源，联合作战

#### 九国盟
- **成员**: 九个小国
- **宗旨**: 抱团取暖，对抗大国
- **福利**: 资源互补，联合防御

### 0.53 国家战争系统

#### 战争触发条件
- 国家关系值≤-8
- 资源争夺（灵脉、秘境）
- 领土纠纷
- 剧情事件（正魔大战）
- 玩家任务触发

#### 战争形式

**0.1. 边境冲突**（小规模）
- **参战**: 边境军队+部分修士
- **目标**: 占领边境城池
- **奖励**: 战功+声望+灵石

**0.2. 局部战争**（中等规模）
- **参战**: 国家主力+门派弟子
- **目标**: 占领数个州郡
- **影响**: 改变势力版图

**0.3. 全面战争**（大规模）
- **参战**: 全国修士+凡人军队
- **目标**: 灭国或逼迫投降
- **影响**: 改变区域格局

#### 玩家参与战争

**参战方式**:
- 加入某国军队
- 作为雇佣修士参战
- 中立旁观（可能影响声望）

**战功系统**:
- 击杀敌方修士: 1-100战功
- 占领城池: 50-500战功
- 破坏敌方设施: 10-50战功

**战功兑换**:
- 兑换灵石、丹药、法宝
- 兑换军功勋章（装备）
- 兑换官职提升
- 兑换特殊功法

### 0.72 国家委托系统

国家可以发布高额委托，招募强力修士完成特殊任务。

#### 示例委托

**S级委托：刺杀敌国长老**
- **委托国**: 风都国
- **目标**: 刺杀天罗国合欢宗长老
- **要求**: 金丹期以上

## 九、数据结构设计

### 0.9099999999999999 国家管理器

```gdscript
class_name 国家管理器

# 所有国家数据
var 国家列表: Dictionary = {}  # {国家ID: 国家数据}

# 国家关系矩阵
var 国家关系: Dictionary = {}  # {国家A_ID: {国家B_ID: 关系值}}

# 联盟列表
var 联盟列表: Array = []

signal 国家事件触发(国家ID, 事件类型)
signal 国家关系改变(国家A_ID, 国家B_ID, 新关系值)
signal 战争爆发(进攻方ID, 防守方ID)

func _init():
	加载国家配置()

func 加载国家配置():
	"""从JSON加载所有国家配置"""
	var 配置文件 = FileAccess.open("res://数据/国家配置/国家列表.json", FileAccess.READ)
	if 配置文件:
		var json = JSON.parse_string(配置文件.get_as_text())
		for 国家数据 in json:
			var 国家 = 国家数据.new()
			国家.从字典加载(国家数据)
			国家列表[国家.国家ID] = 国家
		配置文件.close()

func 获取国家(国家ID: int) -> 国家数据:
	"""获取指定国家数据"""
	return 国家列表.get(国家ID)

func 获取玩家所在国家(玩家: 玩家数据) -> 国家数据:
	"""根据玩家位置获取所在国家"""
	var 玩家位置 = 玩家.当前位置
	for 国家ID in 国家列表:
		var 国家 = 国家列表[国家ID]
		if 国家.是否在国境内(玩家位置):
			return 国家
	return null

func 获取国家关系(国家A_ID: int, 国家B_ID: int) -> int:
	"""获取两国关系值"""
	if not 国家关系.has(国家A_ID):
		return 0
	return 国家关系[国家A_ID].get(国家B_ID, 0)

func 改变国家关系(国家A_ID: int, 国家B_ID: int, 变化值: int):
	"""改变两国关系"""
	if not 国家关系.has(国家A_ID):
		国家关系[国家A_ID] = {}
	if not 国家关系.has(国家B_ID):
		国家关系[国家B_ID] = {}

	var 当前关系 = 获取国家关系(国家A_ID, 国家B_ID)
	var 新关系 = clamp(当前关系 + 变化值, -10, 10)

	国家关系[国家A_ID][国家B_ID] = 新关系
	国家关系[国家B_ID][国家A_ID] = 新关系

	国家关系改变.emit(国家A_ID, 国家B_ID, 新关系)

	# 检查是否触发战争
	if 新关系 <= -8:
		检查战争触发(国家A_ID, 国家B_ID)

func 检查战争触发(国家A_ID: int, 国家B_ID: int):
	"""检查是否触发战争"""
	var 国家A = 获取国家(国家A_ID)
	var 国家B = 获取国家(国家B_ID)

	if 国家A.战争状态 or 国家B.战争状态:
		return  # 已在战争中

	# 触发战争
	开始战争(国家A_ID, 国家B_ID)

func 开始战争(进攻方ID: int, 防守方ID: int):
	"""开始国家战争"""
	var 进攻方 = 获取国家(进攻方ID)
	var 防守方 = 获取国家(防守方ID)

	进攻方.战争状态 = true
	防守方.战争状态 = true

	战争爆发.emit(进攻方ID, 防守方ID)

	# 全服公告
	游戏管理器.显示消息.emit(
		"【国家大事】%s 向 %s 宣战！战争爆发！" % [进攻方.国家名称, 防守方.国家名称],
		"系统"
	)

func 增加玩家国家声望(玩家: 玩家数据, 国家ID: int, 声望值: int):
	"""增加玩家在某国的声望"""
	if not 玩家.国家声望.has(国家ID):
		玩家.国家声望[国家ID] = 0

	玩家.国家声望[国家ID] += 声望值

	# 检查声望等级变化
	var 旧等级 = 计算声望等级(玩家.国家声望[国家ID] - 声望值)
	var 新等级 = 计算声望等级(玩家.国家声望[国家ID])

	if 新等级 != 旧等级:
		var 国家 = 获取国家(国家ID)
		游戏管理器.显示消息.emit(
			"你在 %s 的声望等级提升为：%s" % [国家.国家名称, 新等级],
			"系统"
		)

func 计算声望等级(声望值: int) -> String:
	"""根据声望值计算等级"""
	if 声望值 < -500:
		return "敌视"
	elif 声望值 < -100:
		return "仇恨"
	elif 声望值 < 0:
		return "冷淡"
	elif 声望值 < 100:
		return "中立"
	elif 声望值 < 300:
		return "友善"
	elif 声望值 < 500:
		return "尊敬"
	elif 声望值 < 1000:
		return "崇拜"
	else:
		return "传说"

func 检查国家事件():
	"""定期检查并触发国家随机事件"""
	for 国家ID in 国家列表:
		var 国家 = 国家列表[国家ID]

		# 随机事件概率检定
		if randf() < 0.005:  # 0.5%概率每天
			触发随机国家事件(国家)

func 触发随机国家事件(国家: 国家数据):
	"""触发随机国家事件"""
	var 事件池 = ["灵脉喷发", "秘境开启", "妖兽潮", "天降祥瑞"]
	var 事件 = 事件池[randi() % 事件池.size()]

	国家事件触发.emit(国家.国家ID, 事件)

	# 根据事件类型执行不同逻辑
	match 事件:
		"灵脉喷发":
			国家.灵气浓度临时加成 = 0.01
		"秘境开启":
			国家.临时秘境开放 = true
		# ... 其他事件处理
```

## 十二、平衡性建议

### 1.21 国家实力平衡

- 大国不应完全碾压小国，小国可通过联盟抗衡
- 玩家实力提升后，应能影响国家格局
- 魔道国家虽然强大，但应有相应代价

### 1.22 声望平衡

- 声望获取速度应适中，避免过快或过慢
- 高声望应有实质性好处，但不能过于强大
- 负面声望应有挽回机制

### 1.23 战争平衡

- 战争不应过于频繁，避免玩家厌烦
- 玩家参战应有实质性影响，而非纯粹陪跑
- 战争结果应有多种可能，避免固定结局
