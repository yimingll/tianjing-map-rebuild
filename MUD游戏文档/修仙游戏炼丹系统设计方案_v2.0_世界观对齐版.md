## 一、炼丹概述

### 1.1 何为炼丹?

**白眉老者说书 (云来客栈)**:

> "诸位客官,今日老夫讲的,是'炼丹'二字。"
>
> "何为炼丹?取天地灵药,借丹炉地火,以修士灵力,炼制成丹。"
>
> "丹药者,修士之命根也。"
> "筑基需筑基丹,结丹需结丹丹,凝婴需凝婴丹。"
> "没有丹药,修为难进;有了丹药,突破容易。"
>
> "但老夫要说:丹药虽好,不可多用。"
> "丹毒积累,反噬自身;依赖丹药,根基不稳。"
>
> "修行之道,丹药是助力,不是全部!"

### 1.2 灵药谷 - 青玄界炼丹圣地

**位置**: 中州南部,藏于深山之中

**谷主**: 药仙·葛青云 (化神中期)

**镇谷之宝**: 万药鼎,可炼天地万物为丹

**弟子规模**:
- 内门弟子: 500人
- 外门弟子: 3000人

**特色**: 青玄界最好的丹药都出自灵药谷

**真实案例**:
```
【葛青云的炼丹术】
灵药谷谷主葛青云,化神中期修士,炼丹宗师。
他炼制的筑基丹,成丹率九成,极品率三成。
一颗筑基丹,白云坊市售价1200上品灵石。

江湖人都说:"葛谷主炼的丹,药到病除。"
各大宗门想要好丹药,都要来灵药谷求。

【筑基丹的天价】
散修张三,攒了十年灵石,终于买到一颗筑基丹。
他用1000上品灵石(等于10万中品,1000万下品),买了一颗中品筑基丹。

张三感叹:"这辈子的积蓄,就这一颗丹了!"
"要是失败,我就再也买不起第二颗了。"

【丹药的重要性】
玄天宗外门弟子李云,练气十三层圆满,想突破筑基。
宗门给他一颗筑基丹,让他闭关。

三天后,李云成功筑基!

散修王铁匠,也是练气十三层圆满,但买不起筑基丹。
他硬闯筑基,结果失败,修为跌落到练气十层。

一丹之差,天壤之别!

## 六、炼丹流程

### 6.1 炼丹准备

1. **选择丹方**: 确定要炼制的丹药
2. **准备药材**: 按丹方要求准备主药和辅药
3. **选择丹炉**: 根据丹药品阶选择合适的丹炉
4. **选择地火**: 炼丹地点的地火品质影响成功率
5. **调整状态**: 修士自身状态影响炼丹成功率

### 6.2 炼丹过程

**第一步: 预热丹炉** (10秒)
- 控制火力大小
- 让丹炉达到合适温度

**第二步: 投入药材** (15秒)
- 按顺序投入主药和辅药
- 先主后辅,先轻后重

**第三步: 炼制** (30秒)
- 控制火候
- 用灵力催动丹炉
- 让药材充分融合

**第四步: 凝丹** (10秒)
- 注入灵气
- 将药液凝聚成丹

**第五步: 收丹** (自动)
- 开炉取丹
- 若触发丹劫,需应对天劫

### 6.3 影响成功率的因素

**炼丹师等级**: 等级越高,经验越丰富,成功率越高

**丹炉品质**: 好丹炉提升成功率

**药材品质**: 优质药材提升成功率

**地火品质**: 灵火、地火提升成功率

**修士状态**: 伤势、丹毒、心境都会影响

**真实案例**:
```
【葛青云炼制筑基丹】
葛青云用万药鼎炼制筑基丹:
- 炼丹师等级: 宗师 (+50%成功率)
- 丹炉: 万药鼎 (+30%成功率)
- 药材: 极品筑基草 (+10%成功率)
- 地火: 灵药谷地心火 (+20%成功率)
- 修士状态: 完美 (+10%成功率)

最终成功率: 基础40% + 120% = 100%以上
结果: 必定成功,还有30%几率出极品!

【散修李铁匠炼制筑基丹】
散修李铁匠想炼筑基丹:
- 炼丹师等级: 学徒 (+0%)
- 丹炉: 玄铁炉 (+5%成功率)
- 药材: 普通筑基草 (+0%)
- 地火: 凡火 (+0%)
- 修士状态: 紧张 (-10%成功率)

最终成功率: 基础40% - 5% = 35%
结果: 大概率失败,白白浪费800上品灵石!

李铁匠最后还是放弃了自己炼,花1000上品买了一颗成品。
```

## 七、丹劫系统

### 7.1 何为丹劫?

高阶丹药炼成时,会引发天劫,称为"丹劫"。

**白眉老者说书**:

> "诸位可知,炼制高阶丹药,会引发天劫?"
>
> "这劫,不是渡给修士的,是渡给丹药的!"
>
> "丹药夺天地造化,逆转生死,天道不容。"
> "故降下劫雷,要将丹药毁灭。"
>
> "炼丹师若想保住丹药,就要用修为和法宝护丹。"
> "成功了,丹药品质提升;失败了,丹药化为灰烬。"
>
> "这便是'丹劫'!"

### 7.2 丹劫触发

| 丹药品阶 | 丹劫概率 | 劫雷数量 |
|---------|---------|---------|
| 一至二品 | 不触发 | 无 |
| 三品 | 10% | 1道 |
| 四品 | 30% | 3道 |
| 五品 | 60% | 6道 |
| 六品及以上 | 100% | 9道 |

### 7.3 丹劫应对

**选项一: 护丹**
- 用修为和法宝保护丹药度劫
- 成功: 丹药品质提升1-2级
- 失败: 丹药被毁,炼丹师可能受伤

**选项二: 放弃**
- 不保护丹药,让其自行度劫
- 成功率极低(<5%)
- 成功则丹药自动升品为"劫丹"(效果+100%)

**真实案例**:
```
【葛青云护丹】
葛青云炼制凝婴丹(五品),触发六道丹劫。
他祭出万药鼎,用化神期修为护丹。

六道劫雷轰然而下!
第一道,万药鼎轻松挡住。
第二道,葛青云催动灵力。
第三道,葛青云开始吃力。
...
第六道,葛青云吐血,但终于挡住!

丹劫过后,凝婴丹从中品提升至上品!
此丹后来被拍卖,售价五万上品灵石。

【散修炼丹引发丹劫】
散修王药师,侥幸炼出一颗三品丹药。
结果触发丹劫(10%概率)!

他只有筑基期修为,根本护不住丹。
一道劫雷下来,丹药化为灰烬。

王药师欲哭无泪:"材料啊!我的材料啊!"
```

## 十、丹毒系统

### 10.1 何为丹毒?

长期服用丹药,会在体内积累丹毒。

**白眉老者说书**:

> "丹药虽好,不可多用!"
>
> "凡丹三分毒,长期服用,丹毒积累。"
> "丹毒多了,修炼速度下降,突破成功率降低。"
> "严重的,还会走火入魔,丧命当场!"
>
> "老夫见过一个散修,仗着有钱,天天嗑丹药。"
> "才五十岁,体内丹毒就积累到极限。"
> "最后走火入魔,爆体而亡,连全尸都没留下!"
>
> "所以老夫劝诸位:丹药是辅助,不是依赖!"

### 10.2 丹毒影响

**丹毒<30点**: 无影响
**丹毒30-60点**: 修炼速度-10%
**丹毒60-90点**: 修炼速度-30%,突破成功率-10%
**丹毒>90点**: 修炼速度-50%,突破成功率-30%,有几率走火入魔

### 10.3 清除丹毒

**方法一: 化毒丹**
- 专门清除丹毒的丹药
- 一颗化毒丹清除20点丹毒
- 价格: 50上品灵石

**方法二: 炼体排毒**
- 通过炼体功法,将丹毒排出体外
- 缓慢但安全

**方法三: 渡劫清除**
- 渡劫时,劫雷会清除体内杂质
- 包括丹毒

**真实案例**:
```
【依赖丹药的教训】
玄天宗有个核心弟子,名叫赵富贵。
他家里有钱,天天嗑丹药提升修为。

五十岁就筑基,百岁就金丹,修炼速度惊人。
但他体内丹毒也积累到90点。

某次闭关突破时,赵富贵走火入魔,差点丧命。
幸亏云无极出手相救,才捡回一条命。

从此赵富贵不敢再依赖丹药,老老实实修炼。
但他的根基已经受损,终其一生也只到金丹后期。

【葛青云的忠告】
葛青云常说:"丹药是助力,不是全部。"
"每月服用丹药不超过三颗,才是正道。"

灵药谷弟子都遵守这个规则,根基稳固,很少走火入魔。
```

## 十二、特色玩法

### 12.1 炼丹大会

**时间**: 每年一次,在灵药谷举办
**参与者**: 青玄界所有炼丹师

**比赛环节**:
1. 药材识别: 限时识别30种药材

3. 品质比拼: 炼制最高品质丹药

**奖励**:
- 冠军: 传说丹方×1,极品丹炉×1,称号"当年丹王"
- 亚军: 高阶丹方×3,灵宝级丹炉×1

```

### 12.2 自创丹方

高阶炼丹师可以自创丹方。

**流程**:
1. 选择主药(决定基础效果)
2. 选择辅药(增强或改变效果)
3. 实验炼制(可能失败,消耗材料)
4. 成功后记录为专属丹方

**真实案例**:
```
【葛青云自创丹方】
葛青云三百年来,自创了十几种丹方。
其中最著名的是"九转回春丹"。

此丹可让重伤濒死的修士快速恢复,效果远超普通疗伤丹。
成为战场上的救命神药。

各大宗门都想买九转回春丹的丹方,但葛青云不卖。
这是灵药谷的独门秘方。
```
