# 修仙游戏灵宠/坐骑系统设计方案
> **注**: 本方案为纯单机/买断制游戏设计，无充值系统

## 一、系统总览

### 核心设计理念
- **捕捉与契约**：野外捕捉妖兽，签订灵魂契约
- **培养与进化**：喂养、战斗提升等级，血脉觉醒进化
- **战斗伙伴**：灵宠协同作战，提供强力辅助
- **坐骑系统**：骑乘灵兽，地面/飞行移动加速
- **多样化功能**：采集、探测、储物等特殊能力
- **情感养成**：好感度系统，灵宠有独立性格

>
> **重要**: 本文档定义灵宠系统
>
> **依赖系统**: 境界系统、货币系统
>
> **参考标准**: 详见《【重要】文档统一标准.md》

## 四、灵兽获取系统

### 1. 野外捕捉

#### 捕捉条件
- 妖兽血量<30%
- 使用**捕兽符/灵兽袋**
- 捕捉成功率 = 基础成功率 × (1 - 当前血量%) × 品质系数

#### 捕捉成功率
| 妖兽品阶 | 基础成功率 | 品质系数 |
|---------|-----------|---------|
| 一阶 | 80% | 凡兽×1.0 |
| 二阶 | 60% | 低等妖×0.8 |
| 三阶 | 40% | 中等妖×0.6 |
| 四阶 | 25% | 高等妖×0.4 |
| 五阶及以上 | 10% | 灵兽×0.2，神兽×0.05 |

#### 捕兽道具
```
【捕兽符】(一品) - 捕捉一至二阶妖兽
【灵兽袋】(三品) - 捕捉三至四阶妖兽
【困仙索】(五品) - 捕捉五至六阶妖兽
【封神鞭】(七品) - 捕捉七至八阶妖兽（极难获得）
【山河社稷图】(九品) - 可捕捉一切（传说级）
```

### 2. 孵化培养

#### 妖兽蛋获取
- Boss掉落
- 秘境宝箱
- 任务奖励
- 野外妖兽巢穴
- 拍卖行购买

#### 孵化条件
| 蛋品质 | 孵化时间 | 孵化条件 |
|-------|---------|---------|
| 普通蛋 | 24小时 | 无特殊要求 |
| 精良蛋 | 3天 | 需要灵气环境 |
| 稀有蛋 | 7天 | 需要特定属性灵气 |
| 史诗蛋 | 30天 | 需要高浓度灵气+特殊材料 |
| 传说蛋 | 100天 | 需要灵脉+天材地宝 |
| 神话蛋 | 365天 | 需要仙界灵气+神物 |

### 3. 契约驯服

#### 契约类型
**血契（最强绑定）**：
- 主仆血脉相连，主死宠死
- 灵宠100%忠诚，可共享生命
- 只能契约3只
- 无法解除

**魂契（强绑定）**：
- 灵魂链接，心意相通
- 灵宠可感知主人位置和情绪
- 只能契约10只
- 解除需要特殊仪式

**普通契约（弱绑定）**：
- 基础契约，听从命令
- 好感度低时可能叛逃
- 可契约无限只（但同时召唤有限制）
- 可自由解除

### 4. 任务赠送
- 门派入门赠送一只一阶灵宠
- 某些剧情任务赠送特殊灵兽
- 每日任务有几率获得宠物蛋

### 5. 商店购买
- 凡间宠物店：一阶宠物（铜钱购买）
- 坊市灵兽商：二至三阶（灵石购买）
- 拍卖行：稀有灵兽（竞价）

## 五、灵宠培养系统

### 1. 等级成长

#### 经验获取
- **战斗经验**：灵宠参战获得经验（50%玩家经验）
- **喂养经验**：喂食灵兽丹、灵果（固定经验）
- **修炼经验**：灵宠在聚灵阵中修炼（每小时10经验）
- **共同修炼**：主人修炼时灵宠获得10%经验

#### 等级上限
- 灵宠最高等级 = 主人等级 + 10
- 突破品阶后等级清零，属性保留并提升

### 2. 属性成长

#### 基础属性
每级提升：
- 生命值：+5~200（根据品质）
- 攻击力：+1~20
- 防御力：+1~15
- 速度：+1~5
- 灵力：+10~50

#### 资质系统
| 资质等级 | 属性成长 | 获得方式 |
|---------|---------|---------|
| 劣质 | 70%成长 | 凡兽血脉 |
| 普通 | 100%成长 | 低等妖兽 |
| 优秀 | 130%成长 | 中等妖兽 |
| 卓越 | 160%成长 | 高等妖兽 |
| 完美 | 200%成长 | 灵兽血脉 |
| 传说 | 300%成长 | 神兽血脉 |

#### 资质洗练
- 使用**洗髓丹**重置灵宠资质
- 使用**提纯血脉丹**提升血脉品质+1级
- 使用**返祖丹**有几率觉醒上古血脉

### 4. 进化系统

#### 进化条件
1. 达到当前品阶上限等级
2. 收集进化材料
3. 进化成功率检定

#### 进化路线

**普通进化**（线性）：
```
一阶妖兽 → 二阶妖兽 → 三阶妖兽 → ...
```

**分支进化**（多选择）：
```
【灵猫】→ 【云纹豹】（敏捷型）
	   → 【暗影黑豹】（刺客型）
	   → 【剑齿虎】（力量型）
```

**血脉觉醒**（返祖）：
```
【普通蛇】→ 喂养龙血 → 【蛟蛇】→ 【蛟龙】→ 【真龙】
```

**特殊进化**（融合）：
```
【火鸟】+ 【冰鸟】+ 凤凰羽毛 → 【冰火凤凰】
```

#### 进化材料示例
| 进化阶段 | 所需材料 | 成功率 |
|---------|---------|-------|

| 血脉觉醒 | 对应神兽精血×1 + 返祖丹×1 | 10% |

### 5. 好感度系统

| 81-95 | 挚爱 | 战斗属性+20%，解锁合击技 |
| 96-100 | 心灵相通 | 战斗属性+30%，可感知对方情绪 |

#### 好感度获取
- **喂食**：喂灵宠喜欢的食物（+1~+5）
- **抚摸**：每日可抚摸3次（+1/次）
- **战斗**：并肩作战（+0.1/场）
- **治疗**：受伤时及时治疗（+3）
- **赠礼**：赠送珍贵物品（+5~+20）
- **共同修炼**：一起修炼（+0.5/小时）

#### 好感度惩罚
- **忽视**：长期不召唤（-1/天）
- **虐待**：故意让灵宠送死（-20）
- **挨饿**：不喂食（-2/天）
- **背叛**：主人做出违背灵宠性格的事（-10）

## 七、灵宠战斗系统

### 1. 战斗模式

#### 召唤模式
- **单宠战斗**：召唤1只灵宠协战
- **双宠战斗**（金丹期后）：召唤2只灵宠
- **三宠战斗**（元婴期后）：召唤3只灵宠
- **灵宠军团**（化神期后）：召唤5只灵宠

#### 战斗AI设置
- **被动模式**：不主动攻击，只防御
- **防御模式**：优先保护主人
- **平衡模式**：平衡攻防
- **进攻模式**：全力进攻
- **自由模式**：灵宠自主判断
- **手动控制**：玩家直接操控灵宠

### 2. 合击技能

#### 主宠合击
- 主人与灵宠配合释放强力合击技
- 需要好感度≥80
- 消耗主宠双方灵力

#### 多宠合击
- 多只灵宠配合释放组合技能
- 需要特定灵宠组合
- 威力 = 单体技能×宠物数量×协同系数

#### 经典合击示例
```
【龙凤呈祥】= 真龙 + 凤凰
  效果：召唤龙凤幻影，全属性+50%，持续1分钟

【四圣降临】= 青龙 + 白虎 + 朱雀 + 玄武
  效果：召唤四圣虚影，无敌10秒，超强AOE

【元素融合】= 火灵 + 水灵 + 雷灵 + 风灵 + 土灵
  效果：五行合一，制造混沌领域，敌人全属性-50%
```

### 3. 灵宠复活

#### 死亡机制
- 灵宠血量归零后进入濒死状态
- 濒死状态持续30秒，可救治
- 30秒后灵宠死亡，回到灵兽空间

#### 复活方式
- **自动复活**：1小时后自动复活（好感度-10）
- **复活丹**：立即复活，恢复50%生命
- **凤凰涅槃**：凤凰类灵宠自带复活能力
- **召唤师技能**：宠物复活术（立即复活）

## 九、灵宠与其他系统联动

### 1. 与战斗系统
- 灵宠是战斗重要伙伴
- 高阶灵宠战力不亚于修士
- 灵宠可学习主人技能（同步）

### 2. 与炼丹系统
- **灵兽丹**：专门给灵宠服用的丹药
- **妖兽材料**：击败妖兽获得炼丹材料
- **器官入药**：妖兽内丹、精血等是珍贵药材

### 3. 与炼器系统
- **妖兽材料**：皮、骨、鳞片用于炼器
- **灵兽甲**：为灵宠炼制护甲
- **灵兽武器**：为爪牙附魔增强

### 4. 与食物系统
- 灵宠需要进食（饥饿度系统）
- 不同灵宠喜欢不同食物
- 喂食特殊食物可提升属性

### 5. 与种族系统
- 妖族玩家可化为妖兽形态
- 妖族对灵兽亲和度+20
- 某些种族有专属灵宠

### 6. 与门派系统
- 门派有专属灵兽
- 御兽宗专精灵宠培养
- 贡献度兑换稀有灵兽蛋

### 7. 与洞府系统
- 洞府建造**灵兽园**
- 灵兽可在园中休息、繁殖
- 布置聚灵阵加速灵宠成长

## 十、灵宠繁殖系统

### 繁殖条件
1. 两只同种灵兽（或近亲种）
2. 品阶至少三阶以上
3. 好感度≥60
4. 建造**繁殖园**

### 繁殖周期
| 灵兽品阶 | 怀孕期 | 孵化期 |
|---------|-------|-------|
| 三阶 | 7天 | 3天 |
| 四阶 | 15天 | 7天 |
| 五阶 | 30天 | 15天 |
| 六阶及以上 | 60天 | 30天 |

### 后代遗传
- **血脉遗传**：继承父母更高血脉
- **技能遗传**：有几率遗传父母1-2个技能
- **资质遗传**：资质 = (父资质+母资质)/2 ± 随机浮动
- **变异**：0.1%几率发生变异，诞生特殊灵兽

## 十三、平衡性设计

### 1. 灵宠数量限制
- 最多拥有50只灵宠（可扩展）
- 同时召唤1-5只（根据境界）
- 血契限制3只，魂契限制10只

### 2. 灵宠与玩家平衡
- 灵宠战力 = 玩家战力 × 50%-150%（根据品质）
- 灵宠无法完全替代玩家
- 高阶Boss对灵宠有克制技能

### 3. 培养成本
- 灵宠进化消耗大量资源
- 高品质灵兽培养周期长
- 喂养需要持续投入

### 4. 获取难度
- 高品质灵兽极其稀有
- 神兽级灵宠可遇不可求
- 捕捉高阶妖兽需要实力
