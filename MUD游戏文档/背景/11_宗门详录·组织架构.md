# 玄鉴仙录·宗门详录·组织架构

> **"宗门林立，各有章法。有严苛森严者，有自由松散者。但无论如何，都是修士的归宿。"**

---

## 玄天宗详录

### 组织架构

```
【掌门】
  └─ 云无极（化神后期）

【长老会】（决策机构）
  ├─ 大长老（化神初期，主管内务）
  ├─ 二长老（元婴后期，主管执法）
  ├─ 三长老（元婴后期，主管传功）
  ├─ 四长老（元婴中期，主管资源）
  └─ 其他长老共十二位

【峰主】（各峰负责人）
  ├─ 剑峰（主修剑道）
  │   └─ 峰主：云霄（元婴初期，掌门大弟子）
  ├─ 丹峰（炼丹）
  │   └─ 峰主：药无忧（金丹后期）
  ├─ 器峰（炼器）
  │   └─ 峰主：铁匠（金丹后期）
  ├─ 阵峰（阵法）
  │   └─ 峰主：阵无痕（金丹后期）
  └─ 符峰（符箓）
      └─ 峰主：符天成（金丹后期）

【执事堂】（日常管理）
  ├─ 任务堂（发布任务）
  ├─ 刑罚堂（惩罚违规弟子）
  ├─ 藏经阁（管理典籍）
  ├─ 灵田司（管理灵田）
  └─ 外务堂（对外事务）

【弟子】
  ├─ 亲传弟子（3人，掌门亲自教导）
  ├─ 内门弟子（3000人，金丹以下）
  └─ 外门弟子（20000人，筑基以下）
```

### 入门条件

**外门弟子**：
- 年龄：5-15岁
- 灵根：三灵根以上
- 测试：通过基础测试
- 费用：100下品灵石（贫困者可免）

**内门弟子**：
- 从外门晋升：筑基后自动成为内门
- 或直接招收：双灵根以上天才

**亲传弟子**：
- 掌门亲自挑选
- 天灵根或特殊体质
- 或对宗门有重大贡献者

### 晋升制度

**外门→内门**：
- 突破筑基，自动晋升
- 或立下大功，破格晋升

**内门→核心**：
- 金丹期可成为核心弟子
- 享受更多资源

**弟子→长老**：
- 元婴期可申请成为长老
- 需长老会投票通过（半数以上）

### 资源分配

**外门弟子**：
- 每月10下品灵石
- 可领取基础功法
- 可使用公共修炼室

**内门弟子**：
- 每月100下品灵石
- 可领取高级功法
- 有独立洞府
- 每年一颗筑基丹（筑基前）

**核心弟子**：
- 每月1000下品灵石
- 可领取顶级功法
- 洞府在灵脉之上
- 重要资源优先

**长老**：
- 每月10000下品灵石
- 可自由进出藏经阁
- 独立山峰
- 宗门资源优先

### 宗门规矩

**十大铁律**（违者废除修为，逐出宗门）：
1. 不得欺师灭祖
2. 不得泄露宗门机密
3. 不得残杀同门
4. 不得私自叛逃
5. 不得贪污宗门资源
6. 不得勾结魔道
7. 不得做有损宗门名声之事
8. 不得私自传授宗门功法
9. 不得违抗掌门命令
10. 不得背叛宗门

**其他规矩**：
- 同门之间禁止私斗（可以比武，不得下杀手）
- 必须完成宗门任务（每年至少一次）
- 尊师重道
- 团结同门

### 派系斗争

虽然宗门表面和谐，但内部也有派系：

**保守派**（以大长老为首）：
- 主张维持传统
- 反对改革
- 资源分配偏向老弟子

**改革派**（以掌门为首）：
- 主张改革创新
- 给新人更多机会
- 资源分配透明化

**中立派**（大部分长老）：
- 不站队
- 谁对听谁的

目前，掌门占据上风，但保守派也不容小觑。

### 日常生活

**外门弟子**：
- 清晨：打坐修炼
- 上午：听课学习
- 午间：午餐、休息
- 下午：实战训练
- 傍晚：完成宗门任务（杂务）
- 夜晚：自由修炼

**内门弟子**：
- 更多自由时间
- 可以外出历练
- 任务更少但更重要

**核心弟子**：
- 基本自由
- 只需定期回宗门汇报

## 灵药谷详录

### 组织架构

```
【谷主】
  └─ 葛青云（化神中期）

【长老】
  ├─ 炼丹长老（5位）
  ├─ 灵植长老（3位）
  └─ 外务长老（2位）

【堂口】
  ├─ 炼丹堂（炼制丹药）
  ├─ 灵植堂（种植灵药）
  ├─ 医疗堂（治病救人）
  └─ 销售堂（出售丹药）

【弟子】
  ├─ 内门弟子（500人）
  └─ 外门弟子（3000人）
```

### 特色制度

**师徒制**：
- 灵药谷采用严格的师徒制
- 每个弟子都有一位师父
- 师父负责教导，徒弟负责侍奉

**考核制**：
- 每年一次炼丹考核
- 通过者晋升，失败者降级

**积分制**：
- 炼制丹药可获得积分
- 积分可兑换资源

## 血河宗详录（魔道）

### 组织架构

```
【宗主】
  └─ 血无涯（化神中期）

【长老】
  ├─ 血煞长老（元婴后期）
  ├─ 血影长老（元婴后期）
  └─ 其他长老（8位）

【护法】（打手）
  ├─ 左护法（金丹期）
  ├─ 右护法（金丹期）
  └─ 四大护法（金丹期）

【堂口】
  ├─ 血池堂（修炼血功）
  ├─ 杀戮堂（执行任务）
  └─ 情报堂（收集情报）

【弟子】
  ├─ 核心弟子（200人）
  └─ 普通弟子（2000人）
```

### 魔道规矩

**血河宗铁律**：
1. 强者为尊
2. 不得背叛宗门
3. 完成任务，否则死
4. 不得私自泄露宗门功法
5. 不杀同门（私斗允许，但不能杀）

**弱肉强食**：
- 血河宗内部竞争激烈
- 强者可以抢夺弱者资源
- 但宗主会出面制止过分行为

**任务制**：
- 每个弟子每年必须完成任务
- 任务包括：杀人、抢夺、收集血液等
- 完成任务可获得奖励

## 散修联盟详录

### 组织架构

```
【盟主】
  └─ 独孤求败（元婴后期）

【副盟主】（3位）
  ├─ 毒医蓝凤凰（元婴初期）
  ├─ 阵法大师诸葛孔明（元婴初期）
  └─ 刀客李寻欢（元婴初期）

【长老】（无固定人数，实力达到即可）
  └─ 金丹期以上

【执事】（处理日常事务）
  ├─ 任务执事
  ├─ 仲裁执事
  └─ 外交执事

【成员】
  └─ 所有散修（登记注册即可）
```

### 联盟特点

**松散组织**：
- 散修联盟非常松散
- 成员可以自由进出
- 没有强制任务

**互助为主**：
- 成员之间互帮互助
- 发布任务，其他成员接取
- 报酬自行商定

**保护机制**：
- 如果散修被宗门欺负，可以向联盟求助
- 联盟会出面交涉

## 结语

每个宗门，都有自己的组织架构和规矩。

有的严格，有的宽松。
有的注重实力，有的注重资历。

但无论如何，宗门都是修士的归宿。

在宗门中，你可以：
- 学习功法
- 获得资源
- 结识同道
- 得到保护

当然，也要遵守规矩，完成任务。

选择加入哪个宗门，是修士人生的重要决定。
