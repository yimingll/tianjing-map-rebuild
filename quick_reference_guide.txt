常用出行路线
============================================================================

1. 【从南门进城】
   南门外 → 南门通道 → 南门内广场 → 御街
   → 到商业区买东西
   → 继续北上到皇城/官府

2. 【逛东城白家老宅】
   东城医馆 ← 大宅街 → 白家老宅
   └ 白家老宅内有12个房间可探索

3. 【访问西城区】
   西大街(枢纽) → 西市 | 作坊街 | 民居区 | 贫民区

4. 【进入皇城中心】
   方法A: 南门 → 御街 → 宫前广场
   方法B: 西城 → 西长安街 → 官府 → 皇城
   方法E: 东城 → 东长安街 → 官府 → 皇城

5. 【登上城墙防御线】
   南门通道 → 南城墙 → 城角 → 继续绕城

6. 【出城游历】
   南: 南门 → 南郊官道 → 农田
   北: 北门 → 北郊官道 → 边界
   东: 东门 → 东郊官道 → 海域
   西: 西门 → 丝路起点 → 西漠

7. 【逛贫民区】
   西大街 → 贫民区入口 → 主巷 → 各处
